# 의료기기 인허가 전문가 양성 커리큘럼

> 대상: 인허가 신규 담당자 (엔지니어 또는 QA/RA 배경)
> 목표: 자사 X-ray system(Detector / Handheld Source / GUI SW) 인허가 독립 수행 역량 확보
