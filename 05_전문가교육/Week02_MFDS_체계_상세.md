# Week 2: 국내 MFDS 체계 상세

> **대상**: X-ray 인허가 담당자 (RA 전문가 교육 Week 2/12)
> **소요 시간**: 이론 3시간 + 실습 1시간 + 평가 30분

---

## 1. 학습 목표

이 모듈을 마치면 다음을 할 수 있어야 한다.

1. 의료기기 관련 법령 체계(법률-시행령-시행규칙-고시)를 계층별로 설명한다.
2. 4개 등급 판단 기준을 적용하여 자사 3개 제품의 등급을 결정한다.
3. 허가·인증·신고 각 경로의 제출 서류·처리 기간·심사 기관을 구분한다.
4. GMP 적합인정 절차와 기술문서 심사의 연계 구조를 이해한다.
5. MFDS·FDA·MDR 3지역 허가 체계를 비교하여 동시 제출 전략 수립의 기초를 세운다.

## 2. MFDS 규제 법령 체계

### 2.1 법령 계층 구조

```
[법률] 의료기기법 (법률 제20028호, 최종개정 2024-01-09)
  ├─ [시행령] 의료기기법 시행령 (대통령령)
  │     └─ 등급분류위원회 구성, 수입 통관 특례 등
  ├─ [시행규칙] 의료기기법 시행규칙 (총리령)
  │     └─ 허가·인증·신고 서류, 처리기간, GMP 적합인정 등
  └─ [고시/가이드라인] (식약처 고시)
        ├─ 의료기기 허가·신고·심사 등에 관한 규정 (고시)
        ├─ 의료기기 품목 및 품목별 등급에 관한 규정 (고시)
        ├─ 의료기기 제조 및 품질관리기준 (KGMP 고시)
        └─ 의료기기 기술문서 작성 가이드라인 (가이드)

[법률] 디지털의료제품법 (법률 제19691호, 시행 2025-01-24)  ← 신설
  └─ 디지털의료기기 정의·분류·허가 특례·NIDS 역할 부여
```

**핵심 포인트**: 디지털의료제품법(2025-01-24 시행)은 의료기기법과 **별도 법률**이다. AI/ML 기반 진단 SW 또는 독립형 SaMD는 이 법 적용 여부를 먼저 판단해야 한다.

### 2.2 주요 소관 기관

| 기관 | 역할 |
|------|------|
| 식품의약품안전처 (MFDS) | 허가·인증·신고 최종 결정, GMP 감독 |
| 한국의료기기안전정보원 (NIDS) | 기술문서 심사 위탁, 디지털의료기기 GMP 적합판정 (2025-01-24~) |
| 한국산업기술시험원 (KTL) | GMP 품질관리 심사 위탁 |
| 한국기계전기전자시험연구원 (KTR) | GMP 품질관리 심사 위탁 |
| 한국화학융합시험연구원 (KTR 자회사) | GMP 품질관리 심사 위탁 |

## 3. 의료기기 등급분류 체계

### 3.1 4등급 판단 기준 (의료기기법 §3 + 품목등급 고시)

| 등급 | 잠재적 위해성 | 인허가 유형 | 처리기간 |
|------|-------------|------------|---------|
| 1등급 | 거의 없음 (비침습·에너지 전달 없음) | **신고** | 즉시 |
| 2등급 | 낮음 (단기 피부 접촉·저에너지) | **인증** | 30일 |
| 3등급 | 중간 (에너지 전달, 단기 삽입, 방사선) | **허가** | 65일 (임상 80일) |
| 4등급 | 높음 (영구 이식, 심장·중추신경계 접촉) | **허가** | 65일 (임상 80일) |

**등급 판단 4대 요소**:
1. 인체 접촉 기간 (일시·단기·장기·영구)
2. 침습 정도 (비침습·체표·체강·혈관계·중추)
3. 약품·에너지 전달 여부
4. 생물학적 영향 여부

### 3.2 자사 3개 제품 등급 맵핑

| 제품 | 품목명 (예시) | 등급 | 근거 | 인허가 유형 |
|------|-------------|------|------|------------|
| **X-ray Detector** | 디지털 방사선 영상 검출기 | **3등급** | 방사선 에너지 수신·변환, 진단 영상 생성 → 중등도 위해성 | 허가 (65일) |
| **Handheld X-ray Source** | 휴대용 X선 발생장치 | **3등급** | 방사선 발생·인체 조사 → 중등도 위해성; 일부 고출력 모델 4등급 검토 필요 | 허가 (65일) |
| **촬영실 GUI SW** | 의료영상 획득·처리 소프트웨어 | **2~3등급** | 독립형 SaMD: 진단 보조 기능 유무에 따라 2등급(데이터 전송) 또는 3등급(진단 의사결정 지원) | 인증 또는 허가 |

> **[검증 필요]** 자사 GUI SW의 구체적 기능(진단 보조 여부)에 따라 등급이 달라진다. 디지털의료제품법 적용 가능성도 별도 검토 필요.

### 3.3 품목코드 조회 방법

1. 의료기기 안심책방 → [품목 정보 검색](https://emedi.mfds.go.kr) 접속
2. 제품명·기능 키워드 입력
3. 해당 품목분류번호(7자리) + 등급 확인
4. 동등 품목의 기허가 현황 확인 → 비교 제품 목록 작성 (FDA SE와 유사 개념)

## 4. 허가·인증·신고 프로세스

### 4.1 전체 플로우 (3등급 허가 기준)

```
[사전 준비]
  제조원/수입원 등록 → 품목코드 확정 → GMP 준비 착수
        ↓
[기술문서 작성] ← Step별 가이드라인 참조
  사용목적·작용원리·성능·안전성·원재료·공정도·기준시방서·첨부문서
        ↓
[GMP 적합인정 신청] (병행 진행 권장)
  품질관리심사기관(KTL·KTR 등) → 현장조사 → GMP 적합인정서 발급
        ↓
[허가 신청] (emedi.mfds.go.kr)
  온라인 접수: 기술문서 + GMP 적합인정서 + 수수료
        ↓
[NIDS 기술문서 심사] (위탁)
  안전성·유효성 검토 → 보완 요청 발생 시 clock-stop
        ↓
[MFDS 최종 허가] → 허가증 발급
  처리기간: 65일 (임상자료 필요 시 80일; 보완기간 제외)
        ↓
[변경허가/변경인증/변경신고]
  중요 변경: 변경허가 / 경미 변경: 변경인증 or 신고
```

### 4.2 기술문서 구성 (의료기기 허가·신고·심사 등에 관한 규정 별표 기준)

| 항목 | 주요 내용 |
|------|----------|
| ① 사용목적·작용원리 | 적응증, 물리적 작동 메커니즘 |
| ② 성능 | 주요 성능 규격, 국제표준 적합 여부 |
| ③ 안전성 | 전기적 안전성(IEC 60601-1), EMC, 방사선 안전, 생물학적 평가 |
| ④ 원재료 | 핵심 부품·재료 목록 (RoHS·REACH 포함) |
| ⑤ 공정도 | 제조 공정 흐름도 |
| ⑥ 기준·시방서 | 완성품 규격·시험방법 |
| ⑦ 첨부문서 | IFU, 라벨, 유지보수 설명서 |

> **Tip**: MFDS 기술문서 = FDA eSTAR Device Description + Performance Testing 섹션에 대응. MDR Annex II §6.2와도 구조가 유사하여 3지역 공통 기반 문서로 활용 가능.

### 4.3 보완(Deficiency) 처리

- MFDS 보완 요청 발급 시 → 처리기간 clock-stop
- 보완 기간: **3개월** 이내 (연장 요청 가능)
- 보완 미제출 시 허가 신청 반려
- 관련 KB: [MFDS_보완자료_대응전략](../01_규제지식베이스/국내_MFDS/MFDS_보완자료_대응전략.md)

## 5. GMP (의료기기 제조 및 품질관리기준) 체계

### 5.1 KGMP 개요

- 근거: 의료기기법 §6, 시행규칙 §12·별표 1·2 (KGMP 고시)
- 범위: 설계관리(Design Controls), 제조공정, 품질시스템 전반
- ISO 13485:2016과 구조 동일, 한국어 문서·전산화 밸리데이션 등 추가 요건 존재
- 상세 비교: [KGMP_QMSR_ISO13485_비교_통합전략](../01_규제지식베이스/국제표준_IEC_ISO/KGMP_QMSR_ISO13485_비교_통합전략.md)

### 5.2 GMP 적합인정 절차

```
GMP 심사 신청 → 심사기관 배정 → 품질문서 심사 → 제조소 현장조사
→ 결함사항 통보(3개 등급: Critical/Major/Minor)
→ 시정 조치 제출 → 적합인정서 발급 (유효기간 3년, 갱신 필요)
```

**심사기관 6개** (모든 품목군 대상): KTL, KTR, 한국화학융합시험연구원, 한국건설생활환경시험연구원, TÜV SÜD Korea, TÜV Rheinland Korea

**디지털의료기기**: NIDS가 2025-01-24부터 GMP 적합판정 업무 수행

## 6. 3지역 비교 매트릭스 (MFDS vs FDA vs MDR)

| 비교 항목 | MFDS (한국) | FDA (미국) | MDR (EU) |
|----------|------------|-----------|---------|
| 등급 체계 | 1~4등급 | Class I·II·III | Class I·IIa·IIb·III |
| 주요 경로 | 허가/인증/신고 | 510(k)/De Novo/PMA | CE 인증 (NB 심사) |
| 기술문서 명칭 | 기술문서 | eSTAR | Technical Documentation (Annex II) |
| 품질시스템 | KGMP (ISO 13485 기반) | QMSR (21 CFR 820, 2026-02-02) | ISO 13485 + MDR 요건 |
| 심사기관 | NIDS (위탁), MFDS | FDA OHT | NB (제3자 인증기관) |
| 처리기간 (3등급/Class II) | 65일 | 90일 | 60~90일 (NB에 따라 상이) |
| 임상 요건 | 국내 임상시험 or 해외 임상 인정 | 성능시험 데이터 위주 (Class II) | CER (임상평가보고서) 필수 |
| 보완 처리 | 보완 요청 (clock-stop) | AI (Additional Information) | Deficiency Letter (clock-stop) |
| Post-Market | 이상사례 보고 (의료기기법 §31) | MDR 21 CFR 803 | MDR Art. 83 + Vigilance |
| UDI | UDI 포털 (MFDS) | GUDID (FDA) | EUDAMED |
| SaMD 특이사항 | 디지털의료제품법 (2025-01-24) | FDA AI/ML Action Plan | MDR SaMD (MDCG 2019-11) |

### 6.1 자사 3제품 처리 경로 요약

| 제품 | MFDS | FDA | MDR |
|------|------|-----|-----|
| X-ray Detector | 3등급 허가 (65일) | Class II 510(k) (90일) | Class IIa CE (NB) |
| Handheld X-ray Source | 3등급 허가 (65일) | Class II 510(k) + 21 CFR 1020.30 | Class IIb CE (NB) |
| 촬영실 GUI SW | 2~3등급 인증/허가 | Class II 510(k) (SaMD) | Class IIa~IIb (MDR + SaMD MDCG) |

## 8. 실습 과제

### 실습 1: 등급 판단 (30분)

**문제**: 다음 3개 시나리오의 MFDS 등급을 판단하고 근거를 서술하라.

1. 비냉각식 CMOS 기반 X-ray flat-panel detector (FPD), 14×17인치, 무선 전송
2. 휴대형 고전압 발생기 + X-ray 튜브 일체형, 최대 70 kVp, 배터리 구동
3. DICOM Worklist 기반 촬영 프로토콜 관리 SW, 진단 보조 기능 없음

**제출 양식**:
```
제품명: ___
등급: ___등급
근거 (법령 조항·판단 요소): ___
인허가 유형: 허가 / 인증 / 신고
예상 처리기간: ___일
```

### 실습 2: 기술문서 목차 작성 (30분)

X-ray Detector(3등급)의 MFDS 허가용 기술문서 목차를 작성하라. 각 항목에 필요한 시험성적서(국제표준 기반)를 최소 1개 이상 연결하라.

## 9. 평가 기준

### 이론 평가 (20문항, 40점)

| 유형 | 문항수 | 점수 |
|------|-------|------|
| O/X (법령 조항 해석) | 8문항 | 16점 |
| 단답형 (처리기간·기관명·등급) | 8문항 | 16점 |
| 단문 서술 (등급 판단 근거) | 4문항 | 8점 |

### 실기 평가 (2문항, 60점)

| 문항 | 평가 요소 | 점수 |
|------|----------|-----|
| 등급 판단 | 등급 정확성(10) + 근거 적절성(10) + 인허가 경로(5) × 3제품 | 75점 환산 → 30점 |
| 3지역 비교 | MFDS/FDA/MDR 차이 서술 정확성 + 자사 제품 적용 | 30점 |

### 합격 기준

총 100점 중 **70점 이상** 합격. 이론·실기 각각 50% 이상 득점 필수.
