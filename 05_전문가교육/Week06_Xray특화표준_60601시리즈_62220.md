# Week 6: X-ray 특화 표준 — IEC 60601 시리즈 · IEC 62220

> **대상**: X-ray 인허가 담당자 (RA 전문가 교육 Week 6/12)
> **소요 시간**: 이론 3시간 + 실습 1.5시간 + 평가 30분

---

## 1. 학습 목표

이 모듈을 마치면 다음을 할 수 있어야 한다.

1. IEC 60601-1 (일반안전), 60601-1-2 (EMC), 60601-1-3 (방사선방호), 60601-2-28 (X-ray 관구), 60601-2-54 (촬영·투시장치) 각각의 현행 버전·구조·핵심 요구사항을 설명한다.
2. IEC 62220-1 시리즈 (DQE 측정 3종: 1-1 방사선촬영, 1-2 유방촬영, 1-3 동적) 목적·측정량·적용 제품을 구분한다.
3. MFDS·FDA·MDR 3개 지역에서 각 표준의 인정·요구 방식 차이를 비교한다.
4. 자사 3개 제품(X-ray Detector / Handheld X-ray Source / 촬영실 GUI SW)별 적용 표준 목록을 확정한다.
5. 인허가 패키지에서 각 표준의 시험보고서·적합성 선언 작성 방법을 설명한다.

## 2. IEC 60601 시리즈 개요

### 2.1 구조 개념

```
IEC 60601-1  (일반 표준 General Standard)
  ├── IEC 60601-1-2  (보조 표준 Collateral — EMC)
  ├── IEC 60601-1-3  (보조 표준 Collateral — 방사선방호)
  ├── IEC 60601-1-6  (보조 표준 Collateral — 사용성)
  ├── IEC 60601-1-11 (보조 표준 Collateral — 홈케어 환경)
  └── ...
  
IEC 60601-2-28 (개별 표준 Particular — X-ray 관구 조립체)
IEC 60601-2-54 (개별 표준 Particular — 촬영·투시 X-ray 장치)
IEC 60601-2-43 (개별 표준 Particular — 인터벤션 장치)
```

**적용 원칙**: 개별 표준(2-xx)이 일반 표준(1)과 충돌 시 → 개별 표준 우선("deviation" 명시).

### 3.2 IEC 62220-1-1:2015 — DQE 측정 (방사선촬영 정적 검출기)

**DQE(Detective Quantum Efficiency) 정의**

```
DQE(f) = [SNRout(f)]² / [SNRin(f)]²

여기서:
- f: 공간 주파수 (mm⁻¹ 또는 lp/mm)
- SNRout: 검출기 출력 신호대잡음비
- SNRin: 입사 방사선 신호대잡음비
```

**측정 절차 요약 (표준 §6 기준)**

1. **측정 조건 설정**: 관전압(RQA5: 70kVp + 21mm Al 필터), 공기커마 설정
2. **MTF 측정**: 텅스텐 엣지 또는 슬릿카메라 사용 (경사진 엣지 방법)
3. **NPS 측정**: 균일 조사야에서 Noise Power Spectrum 계산 (2D FFT 기반)
4. **DQE 계산**: DQE(f) = MTF(f)² / [WS_norm(f) × q]
   - WS_norm: 정규화 NPS
   - q: 검출기 입사 양자 수 (Air Kerma에서 환산)

**표준 조사 조건 (RQA 스케일)**

| 스케일 | 관전압 (kVp) | 부가 필터 | 적용 |
|---|---|---|---|
| RQA3 | 50 | 10mm Al | 소아 촬영 |
| RQA5 | 70 | 21mm Al | 흉부·복부 표준 |
| RQA7 | 90 | 30mm Al | 대용량 시험 |
| RQA9 | 120 | 40mm Al | 고에너지 |

| Handheld X-ray Source | Type B (배터리 구동, 비접촉) | Type B | Class II (이중절연) 권장 |
| 촬영실 GUI SW | PEMS (소프트웨어 구성요소) | Applied Part 없음 | Class I (호스트 기기 의존) |

## 6. 인허가 패키지 작성 방법

### 6.1 표준 적합성 선언 (Declaration of Conformity / Standards Compliance)

**기술문서 내 포함 항목**

```
표준명: IEC 60601-1:2005+A1:2012+A2:2020
적용 여부: 적용
준수 방법: 시험보고서 참조 (Report No. XXXX-YYYY)
편차(Deviation): 없음 / 있음 → 이유 기술
```

**MFDS 기술문서**: 전기·기계적 안전 공통기준규격 준수 시험성적서 (공인시험기관 발행) 첨부.

**FDA 510(k)**: eSTAR "Standards" 섹션에 각 표준별 적용·비적용·편차 기재.

**MDR Technical Documentation**: Annex II §4 "Standards applied" 표 작성 (EN 번호, 발행일, 적합 여부, 미적합 시 GSPR 조항 대안 근거).

---

### 6.2 DQE 시험보고서 (IEC 62220-1-1 기준)

인허가 패키지에 포함할 DQE 시험보고서 필수 항목:

1. 시험 기관명·인증 번호
2. 시험 장비 목록 (전리함, 선량계, 영상 분석 SW)
3. 시험 조건 (RQA 스케일, Air Kerma 범위, SID)
4. MTF 곡선 (0~Nyquist 주파수)
5. NPS 곡선 (2D 및 1D)
6. **DQE 곡선** (주요 공기커마 레벨 3개 이상)
7. Flat-field 균일도 (%), 잡음 지수 (σ/μ)
8. Acceptance Criteria (사양서 대비 Pass/Fail)

## 8. 실습 과제

### 실습 1: 표준 적용 범위 결정
> 자사 Handheld X-ray Source의 인허가 패키지에 포함할 표준 목록을 작성하라. 각 표준에 대해 (a) 적용/비적용 판단, (b) 해당 시험보고서 필요 여부, (c) 편차(Deviation) 여부를 기재하라.

**기대 산출물**: 표준 적합성 목록 표 (최소 7개 표준 포함)

---

### 실습 2: DQE 시험계획서 작성
> IEC 62220-1-1:2015 기반으로 자사 X-ray Detector의 DQE 측정 시험계획서(Test Plan)를 작성하라. RQA5 조건, Air Kerma 범위 3단계, MTF/NPS/DQE 측정 절차를 포함하라.

**기대 산출물**: A4 2페이지 분량 시험계획서 초안

---

### 실습 3: FDA eSTAR Standards 섹션 기재
> FDA eSTAR에 X-ray Detector의 표준 적합성을 기재하는 방법을 실습하라. IEC 60601-1 Ed. 3.2, IEC 60601-1-2 AMD1, IEC 62220-1-1을 예시로 작성하라.

**기대 산출물**: eSTAR Standards 섹션 샘플 기재 내용
