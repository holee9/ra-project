# Week 4: MDR EU 규제 체계 상세

> **대상**: X-ray 인허가 담당자 (RA 전문가 교육 Week 4/12)
> **소요 시간**: 이론 3시간 + 실습 1.5시간 + 평가 30분

---

## 1. 학습 목표

이 모듈을 마치면 다음을 할 수 있어야 한다.

1. MDR 2017/745의 법적 근거·발효·전환 타임라인을 설명한다.
2. Annex VIII 22개 분류 규칙을 적용하여 자사 3개 제품의 등급(Class I/IIa/IIb/III)을 결정한다.
3. 등급별 적합성 평가 경로(Annex IX/X/XI)를 구분하고 Notified Body 개입 범위를 설명한다.
4. EUDAMED 4대 모듈 의무화 일정(2026-05-28)과 등록 절차를 수행한다.
5. MDR·MFDS·FDA 3지역 인허가 체계를 비교하여 동시 제출 전략의 EU 파트를 수립한다.

## 3. 등급 분류 (Classification)

### 3.1 분류 원칙 (Article 51 + Annex VIII)

- 제조사가 직접 분류 책임 부담 (자가 분류)
- Annex VIII 22개 규칙 순차 적용 → 해당하는 가장 높은 등급 적용
- 의심 시 관할 CA(Competent Authority) 또는 MDCG Manual on Borderline and Classification 참조
  - 최신판: 2023-09 개정 (Manual v2.1.1)

### 3.2 등급 4단계

| 등급 | 위험도 | 예시 |
|------|--------|------|
| Class I | 낮음 | 안경, 수동 휠체어, 반창고 |
| Class IIa | 낮음~중간 | 진단용 X-ray 검출기(Detector), 청진기 |
| Class IIb | 중간~높음 | X-ray 발생장치(Source), 인공호흡기 |
| Class III | 높음 | 심장 판막, 뇌자극기 |

### 3.3 자사 3개 제품 분류 적용

#### (1) X-ray Detector

```
적용 규칙: Annex VIII Rule 10 (Active Therapeutic/Diagnostic Devices)
  → "Devices specifically intended for recording of diagnostic images
     generated by X-ray radiation" → Class IIa

근거 조문: Section 4.4 of Chapter III, Annex VIII
법령 인용: MDR 2017/745 Annex VIII, Rule 10, 4th indent
```

| 항목 | 내용 |
|------|------|
| 등급 | **Class IIa** |
| 적용 규칙 | Rule 10 (진단용 X-ray 영상 기록) |
| 적합성 평가 경로 | Annex IX Ch.I+III (QMS + TD 심사, NB 개입 필수) |
| EMDN 코드 | Q1201040106 (Static X-ray Detector System) 또는 유사 |

#### (2) Handheld X-ray Source

```
적용 규칙: Annex VIII Rule 10
  → "Active devices intended to emit ionising radiation and intended
     for diagnostic and therapeutic interventional radiology including
     devices which control or monitor such devices"
  → Class IIb

근거 조문: Section 4.3 of Chapter III, Annex VIII
법령 인용: MDR 2017/745 Annex VIII, Rule 10, 3rd indent
```

| 항목 | 내용 |
|------|------|
| 등급 | **Class IIb** |
| 적용 규칙 | Rule 10 (이온화 방사선 방출 진단 장치) |
| 적합성 평가 경로 | Annex IX Ch.I+III (일반 제품군당 1개 TD) |
| EMDN 코드 | Q1201040101 (Stationary/Mobile X-ray Unit) 유사 |

> **Class IIb 이유**: Handheld Source는 이온화 방사선을 직접 **방출**하므로 Rule 10 상위 indent 적용 → Class IIb. Detector는 방사선을 방출하지 않고 **기록만** → Class IIa.

#### (3) 촬영실 GUI SW

```
적용 규칙: Annex VIII Rule 11 (Software)
  → "Software intended to provide information which is used to take
     decisions with diagnosis or therapeutic purposes → Class IIa (serious) or IIb (life-threatening)"
  → 촬영 파라미터 제어 + 결과 조회 목적 → Class IIa

[주의] Software가 AI/ML 기반 진단 의사결정 보조를 포함하면 IIb 재검토 필요
근거 조문: Section 6.3 of Chapter III, Annex VIII
```

| 항목 | 내용 |
|------|------|
| 등급 | **Class IIa** (진단 보조 없는 촬영 제어 SW 기준) |
| 적용 규칙 | Rule 11 (Software) |
| 적합성 평가 경로 | Annex IX Ch.I+III |
| 비고 | AI 진단 기능 추가 시 IIb 상향 재검토 |

## 4. 적합성 평가 절차 (Conformity Assessment)

### 4.1 절차 개요 (Article 52)

```
Class I (IS/IM/IR 제외)
  └─ 자가 선언 (Self-Declaration): DoC 발행 + 기술문서 작성
     Notified Body 개입 없음

Class I (IS/IM/IR) → Annex IX Ch.I+II 또는 Annex XI
     NB 개입 (멸균·측정기능·재사용수술기)

Class IIa
  └─ Annex IX Ch.I+III (권장)
       ├─ Ch.I: QMS 심사 (전체 제품군 커버)
       └─ Ch.III Sec.4: TD 심사 (제품 카테고리당 ≥1개 대표 제품)
     또는 Annex X (Type Examination) + Annex XI (Product Conformity)

Class IIb
  └─ Annex IX Ch.I+III (권장)
       ├─ Ch.I: QMS 심사
       └─ Ch.III Sec.4: TD 심사 (제품군(generic group)당 ≥1개)
           [이식형 특수 경우: 모든 기기 개별 TD 심사]
     또는 Annex X + Annex XI

Class III
  └─ Annex IX Ch.I+II (Design Dossier 포함 전체 심사)
     또는 Annex X + Annex XI
```

### 4.2 Annex 역할 정리

| Annex | 내용 | 대상 |
|-------|------|------|
| Annex IX | QMS 심사 (Ch.I) + NB Certificate of Conformity 발급 (Ch.II/III) | IIa, IIb, III |
| Annex X | EU Type Examination (대표 샘플 형식 심사) | IIa, IIb, III (선택 경로) |
| Annex XI | 적합성 선언·제품 적합성 검증 (생산 후 배치 검사 등) | Annex X와 병행 |
| Annex II | 기술문서 (Technical Documentation) 요건 | 모든 CE 기기 |
| Annex III | 사후시장감시(PMS) 기술문서 요건 | 모든 CE 기기 |

### 4.3 Notified Body (NB) 선정

- EU 공인 NB 목록: NANDO 데이터베이스 (https://ec.europa.eu/growth/tools-databases/nando/)
- X-ray 의료기기 대응 주요 NB: BSI (영국·독일), TÜV SÜD, TÜV Rheinland, SGS, Dekra
- NB는 EUDAMED에 인증서 등록 의무 (2026-05-28 의무화)

> **실무 포인트**: NB 선정 시 X-ray (진단방사선) 관련 코드(EMDN, GMDN)에 대한 지정 범위 확인 필수. 적용 범위 외 NB에 신청하면 거부.

## 5. 기술문서 (Technical Documentation)

### 5.1 Annex II 요건 (신규 기기용 TD)

```
[Section 1] Device description and specification
  1.1 Device description (name, model, variants, accessories)
  1.2 Intended purpose, indications, contraindications, target population
  1.3 Principles of operation, scientific rationale
  1.4 Risk class + applicable classification rules (Annex VIII)
  1.5 Novel features

[Section 2] Information to be supplied by the manufacturer
  - Labels, IFU (Annex I Section 23 기준)

[Section 3] Design and manufacturing information
  - Sites, suppliers, critical components, sterilisation (해당 시)

[Section 4] GSPR (General Safety and Performance Requirements)
  - Annex I 전항 대비 적합성 근거 (표준, 시험, 계산)

[Section 5] Benefit-risk analysis + Risk Management (ISO 14971)

[Section 6] Product verification and validation
  - Pre-clinical (bench), clinical (Article 61), software (IEC 62304)

[Section 7] Post-Market Surveillance plan (→ Annex III와 연계)
```

### 5.2 Annex III 요건 (PMS TD)

| 항목 | 내용 |
|------|------|
| PMS Plan | 수집 방법·주기·임계값·PMCF 계획 |
| PSUR | Class IIa/IIb: 최소 2년 주기 보고 |
| PMCF Plan/Report | Annex XIV Part B 기준 |
| PMS Report | Class I: 필요 시 (정기 보고 불필요) |

## 6. EUDAMED 등록

### 6.1 4개 핵심 모듈 (2026-05-28 의무화)

| 모듈 | 의무화 일 | 주요 내용 |
|------|----------|-----------|
| **ACT** (Actor Registration) | **2026-05-28** | 경제적 운영자(제조사·수입사·유통사) 등록, SRN(Single Registration Number) 발급 |
| **UDI/DEV** (Device Registration) | **2026-05-28** (Article 29 기준 2026-01-02) | UDI-DI·UDI-PI 등록, Basic UDI-DI |
| **CERT** (NB & Certificates) | **2026-05-28** | NB 발급 인증서 등록·조회 |
| **SURV** (Market Surveillance) | **2026-05-28** | CA 시장감시 기록 |
| CIR (Clinical Investigations) | 추후 의무화 | 임상시험 등록 |
| VIGL (Vigilance) | 추후 의무화 | 이상사례·FSCA 보고 |

### 6.2 UDI 구조

```
Basic UDI-DI  ←  제품 카테고리 수준 식별자 (EUDAMED 등록 기본 단위)
  ├─ UDI-DI (Device Identifier)  ← 특정 버전·모델 식별
  └─ UDI-PI (Production Identifier)  ← 제조 배치·시리얼·만료일
```

- EU 공인 발행 기관: GS1, HIBCC, ICCBBA, IFA
- Issuing entity 등록 후 GS1 GTIN 기반 UDI 부여 권장

### 6.3 Authorized Representative (AR, Article 11)

- EU 내 주소 없는 제3국 제조사는 **EU 공인 대리인 지정 필수**
- AR은 EUDAMED ACT 모듈에 등록
- 책임: MFDS와 국내 수입자 개념의 혼합 (법적 책임 일부 부담)

## 7. CE 마킹 절차 전체 흐름

```
[1단계] 등급 분류 결정 (Annex VIII)
  ↓
[2단계] 적합 표준 및 GSPR 맵핑 (Annex I)
  ↓
[3단계] 기술문서 작성 (Annex II + III)
  ↓
[4단계] NB 선정 및 계약
  ↓
[5단계] QMS 심사 (Annex IX Ch.I) → NB 방문 심사
  ↓
[6단계] TD 심사 (Annex IX Ch.III 또는 Annex X)
  ↓
[7단계] Certificate of Conformity 수취 (NB 발급)
  ↓
[8단계] EU Declaration of Conformity (DoC) 제조사 발행
  ↓
[9단계] CE Marking 부착 + NB 번호 병기 (Class IIa↑)
  ↓
[10단계] EUDAMED 등록 (ACT + UDI/DEV)
  ↓
[11단계] 시판 후 감시 (PMS) 지속 수행
```

## 9. 자사 3개 제품 EU 인허가 로드맵

### 9.2 Handheld X-ray Source (Class IIb)

| 단계 | 내용 | 비고 |
|------|------|------|
| TD 작성 | Annex II (방사선 방호 강조) | IEC 60601-1-3, 60601-2-28 |

| 항목 | 내용 |
|------|------|

## 11. 실습 과제

### 실습 1: 등급 분류 적용 (30분)

1. MDR Annex VIII 원문(eur-lex.europa.eu) 열람
2. Rule 10 텍스트 전문 읽기
3. 자사 Handheld Source 제품 Intended Purpose 문서에서 Rule 10 해당 indent 특정
4. Class IIb 결정 근거를 1페이지 메모 작성

### 실습 2: NB 선정 체크리스트 (30분)

1. NANDO(https://ec.europa.eu/growth/tools-databases/nando/) 접속
2. "Medical Devices" → 검색 → TÜV SÜD 또는 BSI 선택
3. EMDN 코드 Q1201 계열 지정 여부 확인
4. 확인된 NB 2곳 비교표 작성 (지정 범위, 평균 리드타임, 비용 공개 여부)

### 실습 3: EUDAMED ACT 모듈 샌드박스 탐색 (30분)

1. EUDAMED Playground(https://eudamed.ec.europa.eu/eudamed/play/) 접속
2. 경제적 운영자 등록 시뮬레이션 진행
3. SRN 발급 절차 스크린샷 캡처 후 정리
