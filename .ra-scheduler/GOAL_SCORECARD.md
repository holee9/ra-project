# 목표·진척 스코어카드 (지구최강 RA 시스템)

> 최종 갱신: 2026-06-09 (기준선 수립)
> 목적: 마스터 헌장의 비전("지구상 최강 RA 시스템")을 **측정 가능한 완성 정의(DoD)**로 환산하고, 매 집중 세션마다 진척율·완료시점을 갱신한다.
> 측정은 실측(grep/카운트) 기반. 미측정·미검증은 절대 '완료'로 표기하지 않는다.

## 1. 완성 정의 (Definition of Done)
"지구최강 RA 시스템 = 모든 규제 주장이 1차출처로 검증되고(정확성), 마스터-페르소나-QA게이트로 통제되며(거버넌스), 3지역×3제품이 빠짐없이 매핑되고(완전성), 구조 결함 없이(위생), 실무 공백이 채워지고(공백해소), 변경이 자동 전파되는(고도화) 살아있는 지식·실행 체계."

> ⚠ 측정 주의: 일부 문서가 `[검증 필요]`(대괄호)·`**검증 필요**`(볼드) 혼용 → 지표를 '검증 필요|확정 필요' 광의로 표준화. 또한 세션 중 백그라운드 린터가 로컬 편집본을 truncate하는 현상 발견 → 모든 정정은 원격 원본 재구성+원자 커밋+원격 재검증으로 처리(원격이 권위).

> 해석: 거버넌스·구조(20+10+15%)는 대부분 올라왔으나, **가장 무거운 축인 '검증 정확성(30%)'이 5%**에 불과. 즉 헤드라인 46%는 거버넌스 주도이며, 진짜 '최강'은 검증 그라인드와 공백 데이터가 좌우한다.

## 3. 완료시점 예측 (집중 세션 트랙 B)
**통제 가능 작업(검증·위생·완전성 QA)**
- 검증 완료 13문서. 잔여(광의) 약 29문서. P2 거의 완료(Week11 남음) → 다음 P3 진입문서 헤더. firm 데이터 과제: #104~#108.
- 페이스 가정: 집중 세션 1회당 3~5문서 검증(난이도 혼합). 이번 세션 = 1문서(3태그) 완료(첫 회차 셋업 포함).
- 추정: **검증 축 ≈ 8~12 세션** + 헤더 백필 9문서(≈1 세션) → **트랙 B 통제분 ≈ 10~13 집중 세션**.
- 달력 환산(세션 빈도별): 매일 1세션 → 약 2~3주 / 주 2회 → 약 5~7주 / 주 1회 → 약 2.5~3개월.

**통제 밖(별도 입력 필요) — 100% 완성의 전제**
- 지표5 공백: 협력기관 담당자 연락처·진행현안 실과제 = **지시자 입력 필요**(내 작업으로 못 채움).
- 지표6 벡터 RAG: 임베딩 인프라 필요(후속).

➡ **전망: 트랙 B 검증·정합 완성은 약 10~13 집중 세션. 단 종합 100%는 (a)지시자 데이터 입력, (b)RAG 인프라가 갖춰져야 도달.**
