## 기술문서 섹션 (공통 구조)
1. Device Description (검출기 유형·구조·원리)
2. Intended Use / Indications for Use
3. Classification & Regulatory Status (국내·FDA·MDR 각각)
4. Design Inputs / Outputs (DHF 연계)
5. Risk Management File (**ISO 14971:2019**)
6. Essential Performance / GSPR Checklist
7. Verification & Validation
   - 전기안전 (**IEC 60601-1 Ed 3.2**)
   - EMC (**IEC 60601-1-2:2014+AMD1:2020**)
   - 방사선 (IEC 60601-1-3:2021)
   - 성능 DQE/MTF/NNPS (**IEC 62220-1-1:2015**)
   - SW (IEC 62304:2006/AMD1:2015)
   - Usability (IEC 62366-1)
   - Cybersecurity (IEC 81001-5-1:2021) – 네트워크·무선 해당 시
8. Clinical Evaluation / Equivalence
9. Labeling / IFU (ISO 15223-1:2021 심볼)
10. Post-Market Surveillance Plan
11. Declaration of Conformity
