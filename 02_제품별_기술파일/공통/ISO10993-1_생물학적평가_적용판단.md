# ISO 10993-1 생물학적 안전성 평가 적용 판단 가이드

## 0. 적용 표준 현황

| 표준 | 버전 | 발효 | 비고 |
|---|---|---|---|
| **ISO 10993-1** | **2025 (Ed 6)** | **2025** | 최신판. Ed 5(2018) 대비 주요 개정 |
| ISO 10993-1 | 2018 (Ed 5) | 2018 | FDA 2023 Guidance 기준 참조판 |
| KS P ISO 10993-1 | (국내 부합화) | MFDS 인정 | 국내 적용 |

> **⚠️ 중요**: ISO 10993-1:2025(Ed 6)가 2025년 발효. FDA는 2023-09 최종 지침에서 Ed 5(2018) 기준이나, 향후 Ed 6 전환 예정. MDR/MFDS는 ISO 10993 시리즈 일반 참조 방식이므로 최신판 적용 권장.

## 1. ISO 10993-1:2025 (Ed 6) 주요 개정 사항

### Ed 6 (2025) vs Ed 5 (2018) 핵심 변경

| 항목 | Ed 5 (2018) | Ed 6 (2025) |
|---|---|---|
| 접촉 범주 분류 | Annex A 단일 대형 테이블(Table A.1) | Tables 1~4 (4개 분리) |
| 위험관리 정합성 | ISO 14971 준용 (일부) | ISO 14971과 완전 정합. 용어·원칙·흐름 통일 |
| 누적 노출 시간 | 단일 사용 기준 | 재사용 기기: **누적 총 노출 시간(Total Exposure Period)** 기준 |
| 유전독성(Genotoxicity) | 장기 접촉 시 요구 | **장기(Prolonged ≥24h, Long-term ≥30d) 전 조직에 필수** (intact skin 제외 불가) |
| 발암성(Carcinogenicity) | 이식 기기 주 대상 | Long-term mucosal membrane 접촉 기기로 확대 |
| Local effects 용어 | "Effects after implantation" | "**Local effects after tissue contact**" (비이식 기기 포함) |
| 합리적 예견 오용 | 미규정 | **Reasonably Foreseeable Misuse 고려 필수** |
| 재료 특성분석 우선 | 권고 | 시험 전 기존 데이터·문헌 활용 명시 강화 |

## 2. 제품별 접촉 특성 분석

### 2-1. 접촉 범주 정의 (ISO 10993-1:2025)

**접촉 성격 (Nature of Contact)**

| 분류 | 정의 |
|---|---|
| Surface Device | 신체 표면(피부·점막·상처) 접촉 |
| External Communicating Device | 체내에 간접적으로 연결되는 기기 (혈로 간접 등) |
| Implant Device | 이식 기기 |

**접촉 부위 (Contact Type)**

| 분류 | 예시 |
|---|---|
| Intact Skin | 정상 피부 접촉 |
| Mucosal Membrane | 점막(구강·질 등) |
| Breached/Compromised Surface | 상처·손상 피부 |
| Blood Path Indirect | 혈액 간접 접촉 |
| Tissue/Bone/Dentin | 조직·골·상아질 |
| Circulating Blood | 순환 혈액 직접 접촉 |

**접촉 기간 (Contact Duration)**

| 분류 | 기간 |
|---|---|
| Limited | < 24 시간 (누적) |
| Prolonged | 24 시간 ~ 30 일 (누적) |
| Long-term | > 30 일 (누적) |

---

### 2-2. 제품별 접촉 분류 매트릭스

| 항목 | X-ray Detector | Handheld X-ray Source | 촬영실 GUI SW |
|---|---|---|---|
| **환자 접촉 여부** | ✅ 있음 | ⚠️ 제한적 | ❌ 없음 |
| **접촉 부위** | 피부(Intact Skin) | 피부 일부(콜리메이터 접촉 시) | 해당 없음 |
| **접촉 기기 분류** | Surface Device | Surface Device (접촉부 한정) | 해당 없음 |
| **접촉 기간** | Limited (<24h/회, 누적 평가 필요) | Limited | 해당 없음 |
| **ISO 10993 적용 여부** | ✅ 필수 | ✅ 접촉 부위 필수 | ❌ 적용 제외 |

#### X-ray Detector 접촉 상세
- 환자 신체(흉부·사지·두부 등)가 Detector 전면 카세트/패널에 직접 접촉
- 통상 1회 촬영 수분~수십분 → Limited 범주
- 다수 환자 공용 사용 (재사용) → 누적 총 노출 시간 계산 필요
- 커버 소재(ABS, Carbon fiber 등) 생물학적 평가 대상

#### Handheld X-ray Source 접촉 상세
- 주 접촉: 술자(방사선사·치과의사 등) Intact Skin (손잡이·그립)
- 부 접촉 가능성: 구강 내 삽입형(dental용)이면 Mucosal Membrane → 분류 상향
- 환자 피부 접촉 가능성: 콜리메이터 헤드가 환자 피부에 닿는 경우
- 범용 X-ray Source(non-dental): Surface Device, Intact Skin, Limited
- Dental intraoral: Surface Device, **Mucosal Membrane**, Limited → 시험 요건 강화

#### 촬영실 GUI SW
- 순수 소프트웨어. 환자·사용자 신체와 직접·간접 접촉 없음
- ISO 10993-1 적용 제외 (FDA Guidance: "no direct or indirect tissue contact → no biocompatibility data required")

## 4. 3개 지역 규제 요건 비교

### 4-1. 생물학적 평가 요건 비교 매트릭스

| 항목 | MFDS (국내) | FDA (미국) | MDR (EU) |
|---|---|---|---|
| **적용 기준 표준** | ISO 10993-1 (KS P ISO 10993-1) + 식약처 고시 | ISO 10993-1:2018 (2023 Guidance) | ISO 10993 시리즈 (조화표준 목록) |
| **가이드라인** | 위험관리 기반 의료기기 생물학적 평가 가이드라인 (2021-12) | Use of ISO 10993-1 Final Guidance (2023-09) | MDR Annex I GSPR §10(d), MDCG |
| **제출 위치** | 기술문서 – 비임상시험자료 섹션 | eSTAR – Biocompatibility Section | Technical Documentation – Annex II §6.1 |
| **면제 근거** | 비접촉 기기 면제 가능, 이유 서면 입증 필요 | 비접촉 기기 → 생체적합성 데이터 불필요 명시 | GSPR 비적용 근거 문서화 필요 |
| **기존 데이터 활용** | 동등 소재 데이터 인정 (서면 입증) | 문헌·기존 데이터 우선 활용 명시 | Technical Documentation 내 포함 |
| **시험기관 요건** | GLP 기관 (ISO 10993 시험 공인기관) | GLP 준용, accredited lab 권장 | ISO 17025 또는 GLP |
| **ISO 10993-1:2025 인정** | [검증 필요] 부합화 표준 개정 시 인정 | [검증 필요] FDA guidance 업데이트 필요 | 조화표준 목록 개정 시 공식 인정 |

### 4-2. MFDS 특이사항

- **적용 법령**: 의료기기법 제6조, 의료기기 기준규격(고시) 별표 1 비임상시험항목
- **심사 기준**: 「의료기기의 생물학적 안전에 관한 자료 심사지침」(식약처 고시)
- **위험관리 기반 평가 가이드라인 (2021-12)**: ISO 14971 프레임 내 생물학적 위험 평가 통합 방식 채택
- **비접촉 증명**: 소재·제조공정·임상 사용 조건 문서로 비접촉 근거 입증 시 시험 면제 가능
- **화학적 특성분석(ISO 10993-18)**: 소재 신규 개발 시 분석 자료 병행 요구 가능

### 4-3. FDA 특이사항

- **Final Guidance (2023-09)**: "Use of International Standard ISO 10993-1, 'Biological evaluation of medical devices – Part 1'" — ISO 10993-1:2018(Ed 5) 기준
- **비접촉 면제 원칙**: X-ray 기기가 신체 직접/간접 접촉이 없으면 생체적합성 데이터 불필요 명시
- **eSTAR 제출**: Biocompatibility Section에 (1) 면제 근거 또는 (2) 시험결과 요약 포함
- **기존 소재 활용**: 동일 소재의 시판 기기 데이터, 문헌 활용 권장 (시험 최소화)
- **Biocompatibility Assessment Resource Center**: FDA 온라인 tool 활용 endpoint 결정 가능

### 4-4. MDR (EU) 특이사항

- **MDR Annex I GSPR §10**: 기기 소재의 생물학적 적합성 요건 명시
  - §10(d): 생체적합성 보장 요건 — 조화표준 ISO 10993 시리즈 활용
- **Technical Documentation (Annex II §6.1)**: Pre-clinical/clinical data 섹션에 생물학적 평가 보고서 포함
- **조화표준 목록**: ISO 10993-1, ISO 10993-5, ISO 10993-10, ISO 10993-17, ISO 10993-18 등 다수 수재
- **MDCG guidance**: 생물학적 평가 전용 MDCG 가이던스는 미발행; GSPR §10 해석은 NB/SCENIHR 가이드 참조
- **Notified Body 역할**: NB는 생물학적 평가 보고서(ISO 10993-1 기반) 및 시험 결과 검토

## 5. 생물학적 평가 보고서 (BER) 구성 요소

ISO 10993-1:2025 기준 BER(Biological Evaluation Report) 필수 항목:

```
BER 구성
├── 1. 기기 설명 및 의도된 사용 (Device description & intended use)
├── 2. 소재 식별 (Material identification)
│   ├── 접촉 소재 목록 (접촉 부위별)
│   ├── 화학적 특성분석 요약 (ISO 10993-18)
│   └── 제조 공정 잔류물 평가
├── 3. 접촉 특성 (Contact characterization)
│   ├── 접촉 분류 (Nature / Type / Duration)
│   └── 누적 총 노출 시간 계산
├── 4. 기존 데이터 검토 (Existing data review)
│   ├── 문헌 검색 결과
│   ├── 동등 소재 시판 기기 데이터
│   └── 이전 시험 데이터
├── 5. Endpoint별 평가 (Biological hazard identification)
│   ├── 각 endpoint 적용 여부 결정 및 근거
│   ├── 시험 요구 여부 결정
│   └── 시험 미수행 시 면제 근거 문서화
├── 6. 추가 시험 계획 (Test plan) — 해당 시
├── 7. 시험 결과 요약 — 해당 시
├── 8. 위험 결론 (Biological safety conclusion)
│   └── 잔여 위험 허용 가능 여부 (ISO 14971 기준)
└── 9. 변경 이력 및 갱신 계획
```

## 6. 제품별 실무 적용 로드맵

### 6-1. X-ray Detector

**단계별 수행 사항:**

1. **소재 목록 작성**: 전면 패널(Carbon fiber, ABS 등), 케이스, 핸들, 케이블 피복 등 환자 접촉 가능 부위 전체
2. **접촉 시간 계산**: 1회 노출 시간 × 1일 예상 촬영 횟수 → 연간 누적. Limited vs Prolonged 판정
3. **소재 이력 확인**: OEM 소재 공급사 데이터시트, 기존 생체적합성 시험 결과 수집
4. **기존 데이터 충분성 평가**: BER 섹션 4에 정리
5. **추가 시험 결정**: Cytotoxicity · Sensitization · Irritation (최소), Genotoxicity (Prolonged 시)
6. **시험 의뢰**: GLP/ISO 17025 인증 기관 (국내: KFRI, KTR, 해외: SGS, Nelson Labs 등)
7. **BER 작성 및 서명**: RA 전문가 + 독립 검토자(Toxicologist 권장)

**핵심 고려사항:**
- Detector 커버 소재가 Carbon fiber인 경우: 에폭시 레진 성분 분석 필요 (ISO 10993-18)
- 소독제(IPA, 차아염소산) 저항성 시험과 생체적합성 시험 구분 필요

### 6-2. Handheld X-ray Source

**단계별 수행 사항:**

1. **사용 시나리오 확정**: 일반 방사선(non-dental) vs dental intraoral 용도 구분
2. **접촉 부위 분류**:
   - Non-dental: 술자 손잡이(Intact Skin, Limited) / 환자 피부 콜리메이터 접촉(Intact Skin, Limited)
   - Dental intraoral: 콜리메이터 or aimer cone 구강 삽입 → Mucosal Membrane 분류 → Genotoxicity 추가
3. **배터리/충전 단자 부위**: 환자 비접촉 영역으로 면제 근거 기재
4. **재사용/소독 고려**: 소독 후 잔류물 영향 평가 포함

### 6-3. 촬영실 GUI SW

- ISO 10993 **적용 제외** — 신체 직접·간접 접촉 없음
- eSTAR / MDR Tech Doc / MFDS 기술문서에 **비접촉 면제 근거 1문단 기재**로 충분

  > *"촬영실 GUI SW는 순수 소프트웨어로 환자 또는 사용자의 신체와 직접적·간접적 물리적 접촉이 없음. 따라서 ISO 10993-1에 따른 생물학적 안전성 평가는 적용되지 않음."*
