# MFDS 보완자료 대응 전략·샘플

## 1. 제도 개요

### 1-A. 법적 근거
- **의료기기법 제12조** (허가·인증·신고): 심사 중 보완 요청 근거
- **의료기기 허가·신고·심사 등에 관한 규정** (식품의약품안전처 고시): 보완 절차·기간 세부 규정
- **의료기기 임상시험계획 승인에 관한 규정** (2023-02-14 개정): IND 2차 보완 30일로 연장

### 1-B. 허가 심사 처리 기간 (정상 회차)
| 구분 | 처리 기간 | 비고 |
|---|---|---|
| 허가 (일반) | 65일 | 보완 기간 제외 |
| 허가 (임상 포함) | 80일 | 보완 기간 제외 |
| 인증 | 30일 | NIFDS 위탁 |
| 신고 | 즉시 | - |

> 보완 기간은 처리 기간에서 제외 (clock stops)

---

## 2. 보완 절차 흐름

```
접수 → 기술문서 심사 → 보완 요청 (1차)
                              ↓
                     보완자료 제출 (60일 이내)
                              ↓
                         재심사
                              ↓
              추가 보완 필요? → 2차 보완 요청
                              ↓
                     보완자료 제출 (60일 이내)
                              ↓
                       최종 허가 결정
```

### 2-A. 보완 기간
| 구분 | 1차 보완 | 2차 보완 | 비고 |
|---|---|---|---|
| 품목허가 (일반) | **60일** | **60일** | 미제출 시 취하 간주 |
| 임상시험계획(IND) 승인 | 30일 | **30일** (2023년 개정) | 기존 2차=10일 → 30일로 연장 |
| 품질관리심사 (GMP) | 3개월 이내 | - | 시정 후 재심사 신청 |
| 신속심사 | 30일 | 14일 | 별도 트랙 |

> 연장 요청: 기한 전 서면 신청 가능, 1회 한함 (사유 소명 필요)

---

## 3. 주요 보완 요청 유형 및 대응 전략

### Type 1: 기술문서 보완 (가장 빈번)
**빈번 사유**
- 성능 데이터 불충분 (시험 방법·결과 누락)
- 작용원리·사용목적 기술 불명확
- 위험관리 파일 연계 미흡 (ISO 14971)
- SW 문서 미흡 (IEC 62304 산출물 누락)
- 생물학적 안전성 평가 누락 (ISO 10993-1)

**대응 전략**
1. 보완 요청서 접수 즉시 내용 분류 (Type, 소관 부서, 데이터 필요 여부)
2. 추가 시험이 필요한 항목은 즉시 시험 의뢰 (기간 역산 필요)
3. 보완서 구조: 요청 항목별 번호 대응 + 인용 근거 명시
4. 규격 시험 기관과 보완 기한 내 결과 수령 가능 여부 사전 확인

**자사 3제품 적용**
| 제품 | 주요 보완 예상 항목 |
|---|---|
| X-ray Detector | DQE·MTF 시험 데이터, IEC 62220-1-1 적합성 |
| Handheld X-ray Source | kVp·mA 출력 특성, 방사선 방호 (IEC 60601-1-3) |
| 촬영실 GUI SW | IEC 62304 SW 수명주기 산출물, Cybersecurity (2025.01) |

---

### Type 2: 임상 데이터 보완
**빈번 사유**
- 동등성(Equivalence) 근거 불충분
- 국내 임상 데이터 요구 (혁신의료기기·신개발)
- 문헌 검색 전략 미흡 (PICO 불명확)

**대응 전략**
1. 동등 제품 선정 기준 재검토 → 기술·성능·사용목적 3요소 비교표 재작성
2. 문헌 DB: PubMed, EMBASE, Cochrane → 검색 식 문서화
3. 국내 임상이 불가피한 경우: IRB 승인 → IND 신청 트랙 전환 (기간 대폭 증가)
4. MFDS 사전미팅 활용하여 임상 계획 사전 협의

---

### Type 3: 기재 사항 보완 (라벨링)
**빈번 사유**
- 표시기재 필수 요소 누락 (제조번호, 유효기한, 경고문구)
- 한국어 IFU 작성 미흡
- 디지털의료기기 표시기재 가이드라인 (2025.01) 미반영

**대응 전략**
1. 최신 고시 기준으로 표시기재 항목 재점검
2. 의료기기 표시기재 가이드라인 체크리스트 활용
3. 디지털의료기기: SW 버전, 사이버보안 관련 경고문 추가 여부 확인

## 4. 보완서 작성 원칙 및 샘플 구조

### 4-A. 보완서 필수 구성 요소
```
[보완서 표지]
- 제품명, 모델명
- 허가신청번호 또는 접수번호
- 보완 요청 날짜 및 회차 (1차/2차)
- 제출일

[본문 - 항목별 대응]
보완 요청 항목 1:
  ① 요청 내용 인용
  ② 대응 방법 (조치 사항)
  ③ 근거 자료 (첨부 목록 명시)

보완 요청 항목 2:
  ...

[첨부 자료 목록]
- 첨부 1: 성능 시험 성적서 (기관명, 날짜)
- 첨부 2: 수정된 IFU
- ...
```

### 4-B. 샘플 답변 (기술문서 보완 — 성능 데이터)

> **보완 요청 항목**: X-ray Detector의 DQE(Detective Quantum Efficiency) 측정 데이터 미흡

**대응 내용:**

본 기기의 DQE 측정은 IEC 62220-1-1:2015 기준에 따라 공인 시험기관(○○기술원)에서 실시하였습니다.

측정 조건:
- RQA5 빔 품질 (IEC 62220-1-1 §4.1.2)
- 공간 주파수: 0~2 lp/mm 범위
- 결과: DQE(0) = 0.65 (Pass: ≥0.60)

첨부 1: 시험 성적서 (○○기술원, 2026-04-15) 참조

## 5. 3개 지역 보완 절차 비교

| 항목 | MFDS (Korea) | FDA 510(k) | EU MDR (NB) |
|---|---|---|---|
| 보완 명칭 | 보완자료 요청 | Additional Information (AI) Request | Deficiency Letter / Objection |
| 법적 근거 | 의료기기법 §12, 심사규정 | 21 CFR 807.87(m) | MDR Art.52, NB계약 |
| 보완 기간 | **60일** (1차·2차) | **180일** (자동 부여) | NB 계약 의존 (통상 30~60일) |
| Clock 영향 | 처리기간 제외 (stop) | Review clock stop | NB 심사 clock stop |
| 미제출 시 | 취하 간주 → 재신청 필요 | Notice of Withdrawal (취하) | 심사 종료, 재신청 필요 |
| 연장 가능 여부 | 1회 서면 요청 | 별도 요청 불필요 (180일 자동) | NB와 협의 |
| 보완 횟수 제한 | 명시 없음 (통상 2회) | 제한 없음 (단, 신규 정보 시 신규 접수) | NB 재량 |
| Interactive Review | 없음 (공식 절차) | 가능 (clock 미정지) | NB에 따라 상이 |

## 6. 보완 대응 타임라인 (60일 기준)

| D-Day | 활동 |
|---|---|
| D+0 | 보완 요청서 접수 → 내부 공유, 항목 분류 |
| D+1~3 | 보완 항목 담당자 배정, 시험 의뢰 필요 항목 즉시 발주 |
| D+7 | 1차 보완 계획 수립 (항목별 완료 예정일) |
| D+30 | 보완서 초안 완성 → 내부 검토 |
| D+45 | 최종 보완서·첨부자료 완성 |
| D+50 | 제출 (기한 10일 여유 확보) |
| D+60 | 제출 마감 (최대 기한) |

> 주의: 시험이 필요한 항목은 D+0에 즉시 의뢰하지 않으면 60일 내 완료 불가

## 7. Anti-Patterns (해서는 안 되는 것)

1. **부분 보완 제출 금지**: 일부 항목만 제출 시 "미완성 보완"으로 처리될 수 있음 → 전항목 동시 제출 원칙
2. **기한 내 연장 요청 지연 금지**: 연장 요청은 기한 만료 전에 해야 효력 있음
3. **보완 항목 임의 해석 금지**: 모호한 요청은 MFDS 담당자에게 반드시 확인 후 대응
4. **기존 자료 재제출 금지**: 이미 제출한 자료를 변경 없이 재제출하면 동일 지적 반복 가능
5. **시험 성적서 유효기간 확인 누락 금지**: 제출 시점 기준 유효한 성적서인지 확인

## 8. MFDS 사전 상담 활용 전략

- **MFDS 의료기기종합상담센터**: 1577-1255
- **민원인 사전미팅(Q-Sub 유사)**: 정식 제도화는 없으나 담당 부서 협의 가능
- **혁신의료기기·신개발의료기기**: 식약처 혁신의료기기지원단 활용 권장
- 보완 요청 전 예상 지적 항목은 Pre-submission 상담으로 선제 해소 권장
