# 자사 제품별 등급 분류 (국내 / 미국 / 유럽)

> 본 문서는 공식 분류 기준에 근거한 **초안**. 실제 제품 세부 사양(정격·용도·적용부위·능동/수동 여부 등)에 따라 변동 가능.
> 확정 시 근거 문서(식약처 품목분류 통지, FDA Product Code 검색 결과, MDR Rule 적용 근거)를 첨부할 것.
> **최종 갱신**: 2026-04-22 (웹 리서치 교차검증 반영)

## 1. X-ray Detector (평판 검출기)

### 1.1 국내 (MFDS) — **2등급**
- **근거**: 「의료기기 품목 및 품목별 등급에 관한 규정」
- **관련 품목 예시**: 진단용엑스선촬영장치, 디지털치과진단용구강내엑스선촬영장치, 디지털치과진단용파노라마엑스선촬영장치 등 모두 2등급
- **검증 필요**: 검출기 자체가 독립 품목(구성품)으로 허가되는지, 시스템 일부로 허가되는지 확인 필수.

### 1.2 미국 (FDA) — **Class II**
- **Product Code**: **MQB** (Solid State X-Ray Imager, Flat Panel/Digital Imager)
- **규제 번호**: 21 CFR 892.1680 / 892.1720 계열
- **가이던스**: "Guidance for the Submission of 510(k)s for Solid State X-ray Imaging Devices"
- **심사 경로**: 510(k) Premarket Notification (Predicate Device 기반)

### 1.3 유럽 (MDR 2017/745) — **Class IIa 가능성 높음** ⚠️ 재검토
- **핵심 포인트**: MDCG 해석상 Rule 10은 "능동적으로 이온화 방사선을 방출하는 기기"에 적용.
- **Detector는 수동 수신 기기**: "devices specifically intended for recording of diagnostic images generated by X-ray radiation are classified as class IIa" 원칙 적용 가능 → **Class IIa (Rule 10 후단 또는 Rule 17 계열)**.
- **단, 시스템 일부로 판매 시**: 전체 X-ray 시스템(Source 포함)의 등급(IIb)에 따름.
- **확정 필요**: 독립 판매/OEM 공급 여부, intended purpose 문구에 따라 IIa/IIb 결정.

## 2. Handheld X-ray Source (소형 X선 발생장치)

### 2.1 국내 (MFDS)
- **등급**: **2등급** (대부분의 진단용 X선 발생장치) / 용도·출력에 따라 **3등급** 가능
- **병행 규제**: 「진단용 방사선 발생장치의 안전관리에 관한 규칙」(보건복지부령, 최근 개정 2025-07-18 제1122호)
  - 설치 3일 전 신고, 3년 주기 검사, 안전관리책임자 임명 의무
- **2025-01-24 시행 「디지털의료제품법」** 적용 여부는 제품 특성(AI·SW 포함 여부) 기준 검토 필요

### 2.2 미국 (FDA) — **Class II**
- **Product Code 후보**: **IZL** (Mobile X-ray System) / **EAF** (Stationary X-ray System) / **KPR** (Dental X-ray) – 실제 용도에 따라 선택
- **규제 번호**: 21 CFR 892.1720 (Stationary/Mobile X-ray System)
- **Radiation 성능표준 (필수 준수)**: **21 CFR 1020.30 ~ 1020.33**
  - 누설선량, 조사야, 타이머, 관전압·관전류 정확도 등 정량 요건
  - 조립자 보고서 Form FDA 2579: 2023년 개정(88 FR 3638)으로 **인증 부속 컴포넌트 일부는 제출 의무 완화**됨. 제품별 적용 여부 개별 확인 필요
- **Initial Product Report**: FDA Form 2877 (수입 시) / 신규 제품 최초 보고

### 2.3 유럽 (MDR) — **Class IIb**
- **근거**: Annex VIII Rule 10 – "active devices intended to emit ionizing radiation for diagnostic or therapeutic radiology" = Class IIb
- **NB 심사 필수**: Annex IX QMS + TD Assessment

## 3. 촬영실 GUI 기반 SW (Medical Imaging Software)

### 3.1 국내 (MFDS) — **2등급** (보조 기능 수준에 따라 상향 가능)
- **근거**: 의료용 소프트웨어 허가·심사 가이드라인, 「디지털의료기기소프트웨어 허가·심사 가이드라인」
- **2025년 신법**: 「디지털의료제품법」 2025-01-24 시행 – AI/SW 적용 디지털의료제품 별도 체계 (변경관리계획, 구성요소 성능평가 등)
- **독립형 SW 사용적합성 가이드라인** 별도 적용 (독립형인 경우)

### 3.2 미국 (FDA) — **Class II**
- **Product Code 후보**: **LLZ** (Medical Image Management and Processing System) / **QIH** (Radiological CADe SW for Lesions) / **QDQ** (Radiological CAD Triage SW)
- **510(k) 필수**, eSTAR 의무 (2023-10-01~)
- **Cybersecurity**: 2023-09-27 Final Guidance 적용, **SBOM 법적 의무** (Section 524B, Omnibus 2022)
- **AI/ML 적용 시**: PCCP (Predetermined Change Control Plan) Final Guidance 2024-12-03 활용 가능

### 3.3 유럽 (MDR) — **Class IIa 이상**
- **근거**: Annex VIII **Rule 11** (정보 제공·진단·치료 영향 SW)
- **MDCG 2019-11 Rev.1** (2025-06 발간) 적용
- **일반 원칙**: "nearly all medical device software will be classified as at least Class IIa"
- **AI 기반 영상 판독 보조 SW**: 중증도·영향에 따라 IIa 또는 IIb

## 4. 통합 시스템 판매 시 전략
- 3개 구성품이 하나의 **시스템(X-ray 촬영시스템)**으로 묶여 판매되는 경우:
  - 시스템 등급 = 구성품 최고 등급 (Source 기준 MDR Class IIb, FDA Class II)
  - 기술문서는 시스템 단위 + 구성품별 사양서 병행
- **OEM 공급** 시: 구성품 단독으로도 별도 허가 유지 권장 (다운스트림 대응용)
