# MFDS 디지털의료기기 표시기재 가이드라인 대응

## 1. 개요

| 항목 | 내용 |
|---|---|
| 목적 | MFDS 의료기기 표시·기재 규정(고시 제2024-77호) + 디지털의료제품법 시행규칙(2025) 기반 디지털의료기기 특화 표시기재 요건 정리 |
| 적용 제품 | 촬영실 GUI SW (주) / X-ray Detector·Handheld Source (SW 내장 부분) |
| 기준 법령 | 의료기기 표시·기재 등에 관한 규정 고시 제2024-77호(2024-12-04 시행) / 디지털의료제품법 시행규칙(2025-02-28 시행) |

---

## 2. 규제 체계

### 2.1 근거 법령·고시 계층

```
디지털의료제품법 (법률, 2025-01-24 시행)
  └─ 디지털의료제품법 시행령
  └─ 디지털의료제품법 시행규칙 (2025-02-28 시행)
의료기기법 제24조 (표시기재 사항)
  └─ 의료기기 표시·기재 등에 관한 규정
       (식약처 고시 제2024-77호, 2024-12-04 시행) ← 현행 최신
  └─ 의료기기 표시기재 가이드라인 (민원인안내서, MFDS)
  └─ 디지털의료기기 표시기재 가이드라인 (AI·SW 특성 반영, MFDS 별도 제정)
```

### 2.2 고시 제2024-77호 주요 개정 내용

| 개정 조항 | 변경 내용 |
|---|---|
| 제6조 (주소 기재) | 제조·수입업체 주소 기재 간소화 — 허가증·신고증 주 사업장 주소 1개만 기재 허용 (종전: 전체 주소 강제) |
| 제8조·제9조 (전자민원) | 전자민원시스템명 현행화 ("의료기기전자민원창구") |
| SW·디지털 관련 | 음성·수어 영상 변환코드 관련 조항 신설 (접근성 강화) |
| 디지털의료제품법 연동 | 디지털의료기기 전자 표시·기재(e-IFU) 근거 마련 |

## 5. MFDS 디지털의료기기 GMP 8종 유형군 매핑

> 디지털의료기기 제조 및 품질관리 기준 (2025)에서 분류한 8대 유형군:

| 유형군 | 정의 | 자사 해당 제품 |
|---|---|---|
| 유형 1: 독립형 SW | 하드웨어 없이 독자 운영되는 SW 의료기기 | 촬영실 GUI SW |
| 유형 2: 내장형 SW | HW 의료기기에 내장된 SW | X-ray Detector 펌웨어, Handheld 펌웨어 |
| 유형 3: 모바일 앱 | 모바일 기기에서 동작하는 SW 의료기기 | — |
| 유형 4: AI/ML 기반 SW | AI·ML 알고리즘으로 진단 보조 수행 | 촬영실 GUI SW (CAD 기능 포함 시) |
| 유형 5: SaaS/클라우드 | 클라우드 기반 의료기기 서비스 | — |
| 유형 6: CDSS | 임상 의사결정 지원 시스템 | — |
| 유형 7: 모니터링/측정 SW | 생체신호 모니터링 및 측정 | — |
| 유형 8: 진료 기록 연계 SW | HIS·EMR·PACS 연동 SW 의료기기 | 촬영실 GUI SW (PACS 연동 시) |

## 6. e-IFU (전자 사용설명서) 적용 가이드

| 조건 | 내용 |
|---|---|
| 허용 근거 | 고시 제2024-77호 + 디지털의료제품법 시행규칙 |
| 적용 방법 | ① SW 인터페이스 내 내장 IFU / ② 웹 URL 링크 / ③ QR 코드 |
| 종이 IFU 병행 | 사용자 요청 시 종이 제공 의무 유지 |
| 업데이트 이력 | e-IFU 내 버전 이력·개정 내용 관리 |
| 접근성 | 음성 안내·수어 영상 변환코드 제공 권장 (고시 제2024-77호 신설) |

## 7. 실무 체크리스트 (촬영실 GUI SW IFU 작성 시)

```
□ SW 버전·빌드 번호·릴리스일 기재
□ 지원 OS 목록 및 최소·권장 HW 사양 명시
□ 네트워크 보안 설정 (TLS 버전·방화벽·포트) 기재
□ 사이버보안 사용자 책임 사항 명시 (비밀번호·패치 적용)
□ 알려진 잔류 사이버보안 위험 및 완화 조치 기재
□ SW 업데이트 방법·주기·절차 기재
□ 데이터 백업·복구 절차 기재
□ AI 기능 포함 시: 알고리즘 목적·성능 지표·한계·학습 데이터 기재
□ AI 오작동 대처: "AI 출력은 임상의 최종 판단을 대체하지 않음" 명시
□ 호환 기기 목록 (Detector·X-ray source·PACS 시스템) 기재
□ 디지털의료기기 등급 및 허가번호 기재
□ IFU 버전 관리 (SW 버전 변경 시 IFU 동기화 절차)
□ e-IFU 적용 시: 접근 경로 및 종이 IFU 요청 절차 기재
```
