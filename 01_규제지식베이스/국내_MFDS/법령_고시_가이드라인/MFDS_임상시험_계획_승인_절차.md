# MFDS 임상시험 계획 승인 절차

## 1. 법적 근거

| 법령 | 조항 | 내용 |
|---|---|---|
| 의료기기법 | 제10조 | 임상시험계획 승인 의무 |
| 의료기기법 시행규칙 | 제20조 | 신청 서류·심사 기준 |
| MFDS 고시 제2023-12호 | 전문 | 의료기기 임상시험계획 승인에 관한 규정 (2023-02-14 시행) |
| 의료기기법 시행규칙 | 제24조 | 의료기기 임상시험 관리기준 (KGCP) |

> **원칙**: 의료기기로 임상시험을 하려는 자는 임상시험계획서를 작성하여 **식품의약품안전처장의 승인**을 받아야 함 (의료기기법 §10①).

---

## 2. 승인 면제 대상 (의료기기법 §10① 단서, 시행규칙 §20①)

다음에 해당하는 경우 사전 승인 없이 임상시험 실시 가능:

| 면제 유형 | 요건 |
|---|---|
| 시판 후 임상관찰 | 이미 허가된 의료기기의 허가사항 범위 내 임상적 효과 관찰 |
| 저위해 기기 | 대상자에게 위해를 끼칠 우려가 적은 기기 (총리령 규정) |
| 체외진단 | 체외진단의료기기 임상적 성능시험 (별도 규정 적용) |

> **주의**: 면제 기기라도 기관생명윤리위원회(IRB) 승인은 별도로 필요.

---

## 3. 승인 신청 절차 (MFDS §10 흐름)

```
[사전상담] ─→ [신청서 제출] ─→ [접수·보완 요구] ─→ [심사 (30일)] ─→ [승인/반려]
    ↑                                      ↓
권장 (공식 절차 아님)               보완 기간 30일 (추가 10일 가능)
```

### 3-1. 사전상담 (Pre-consultation)

- 근거: 고시 제2023-12호 내 권장 절차
- MFDS 의료기기심사부 의료기기임상시험팀 직접 신청
- 사전상담일 이전 수시 면담·자료 검토 가능
- 사전상담 후 신청 시 특별한 사유 없으면 **30일 이내 승인** 보장

### 3-2. 신청 채널

| 채널 | 설명 |
|---|---|
| 정부24 (온라인) | gov.kr → "의료기기임상시험계획승인" 검색·신청 |
| 우편·방문 | MFDS 민원실 (오송 본부) |

## 4. 제출 서류 (의료기기법 시행규칙 §20②)

| 서류 | 세부 내용 |
|---|---|
| ① 임상시험계획서 | 안전성·유효성 증명 설계, 대상자 동의·보상 규약 포함 |
| ② 기술문서 | 사용목적, 작용원리, 성능·안전성 시험규격 (비임상시험 성적서 가능) |
| ③ GMP 적합성 증명 | 임상시험용 GMP 적합인정서 **또는** ISO 13485 인증서 |
| ④ 신청수수료 | 의료기기법 시행규칙 별표 수수료 (기기 등급·시험 유형별 상이) |

### 임상시험계획서 필수 포함 항목 (고시 제2023-12호 별표)

- 시험의 목적 및 배경
- 시험 기기 설명 (원리·구성·작동방식)
- 대상 질환·적응증
- 대상자 선정·제외 기준
- 시험 설계 (무작위배정·맹검 여부)
- 1차·2차 평가변수 (Primary/Secondary Endpoint)
- 통계분석 계획 (샘플 수 산정 근거 포함)
- 위험관리 방안 (이상반응 모니터링·보고 절차)
- 기관생명윤리위원회(IRB) 승인 계획
- 대상자 동의서 (Informed Consent Form)
- 보험·보상 계획

---

## 5. 심사 기준 (고시 제2023-12호 §5)

MFDS는 아래 5가지 기준으로 심사:

| 심사 항목 | 주요 확인 내용 |
|---|---|
| 과학적 타당성 | 임상시험 설계·통계방법의 타당성 |
| 안전성 | 전임상(비임상) 시험 결과 기반 대상자 안전 보장 여부 |
| 윤리성 | IRB 승인 계획, 대상자 동의 절차, 취약 대상자 보호 |
| 기술문서 완결성 | 기기 원리·성능·제조품질 관련 자료의 충분성 |
| GMP 적합성 | 임상시험용 기기의 제조·품질관리 기준 준수 |

---

## 6. 심사 기간 및 보완

| 단계 | 기한 |
|---|---|
| 정상 심사 | **30일** (사전상담 후 신청 기준) |
| 보완 요구 | 최대 **30일** (기간 내 미제출 시 추가 10일 부여 가능) |
| 변경승인 | 경미한 변경: 서면 통보. 중요 변경: 재심사 30일 |

> **보완 기간 중 심사 시계 정지**: 보완 기간은 심사 기간에 포함되지 않음.

---

## 7. 변경승인 (의료기기법 §10①, 시행규칙 §20⑤)

임상시험계획 **변경** 시에도 승인 의무:

| 변경 유형 | 처리 |
|---|---|
| 중요 변경 (Primary Endpoint 변경, 대상자 수 20% 이상 증감 등) | 변경승인 신청 → 심사 30일 |
| 경미한 변경 (시험자 교체, 시험기관 추가 등) | 변경신청 → MFDS 수리 통보 |

## 8. 3개 지역 비교 매트릭스

| 항목 | MFDS (한국) | FDA (미국) | EU MDR |
|---|---|---|---|
| **근거 법령** | 의료기기법 §10, MFDS 고시 2023-12호 | 21 CFR Part 812 (IDE) | MDR Art. 62, ISO 14155 |
| **신청 명칭** | 임상시험계획 승인 | IDE (Investigational Device Exemption) | Clinical Investigation Application (CIV) |
| **위험도 분류** | 위해도 면제 여부만 구분 | Significant Risk (SR) / Non-Significant Risk (NSR) | 별도 분류 없음 (Art. 62 적용 기준으로 구분) |
| **규제기관 승인** | MFDS 필수 | SR: FDA 필수 / NSR: IRB만 필요 | 각 회원국 Competent Authority (CA) |
| **심사 기간** | **30일** | **30일** (자동승인 포함) | **37일** (CA 검토 기한) |
| **보완 대응** | 30일 + 추가 10일 | 추가 정보 요청 가능 (30일 연장) | 추가 정보 요청 가능 |
| **IRB/EC 승인** | 별도 필요 (KGCP) | 별도 필요 (21 CFR 56) | Ethics Committee 필수 (동시 진행 가능) |
| **등록 시스템** | 임상시험포털 (CRIS) | ClinicalTrials.gov | EUDAMED (2025년 이후 의무화) |
| **GMP 요건** | 임상시험용 GMP 또는 ISO 13485 | 21 CFR 820 (QSR) 준수 증명 | Annex I GSPR + ISO 13485 |
| **주요 가이던스** | MFDS 고시 2023-12호, KGCP | IDE Guidance (2013), 21 CFR 812 | MDCG 2021-6 Rev.1 (2023-12), ISO 14155:2020 |

## 9. 자사 3개 제품 영향 맵핑

### 9-1. X-ray Detector (디지털 방사선 검출기)

| 항목 | 내용 |
|---|---|
| MFDS 등급 | 3등급 (방사선 조사장치 관련) [검증 필요] |
| 임상시험 승인 필요 여부 | **필요** — 신규 의료기기, 기존 시판 기기 아님 |
| 주요 Endpoint | DQE 성능 vs 참조기기, 진단 정확도 (민감도·특이도) |
| 비임상 선행 과제 | IEC 62220-1-1 DQE 시험, IEC 60601-1 전기안전 |
| 임상시험 설계 권고 | 동등성 시험 (vs 기존 허가 Detector), 방사선 노출량 비교 포함 |

### 9-2. Handheld X-ray Source (휴대용 X선 발생장치)

| 항목 | 내용 |
|---|---|
| MFDS 등급 | 3등급 (방사선 조사장치) |
| 임상시험 승인 필요 여부 | **필요** — Handheld 형태 신규 기기 |
| 주요 위해 인자 | 방사선 피폭 (환자·조작자), 배터리 발화, 오조사 |
| 주요 Endpoint | 촬영 성공률, 방사선 피폭량 (KERMA), 영상 진단 적합성 |
| 비임상 선행 과제 | IEC 60601-2-28, 21 CFR 1020.30 적합성, 산란 방사선 시험 |
| 임상시험 특이사항 | 구강내 촬영 등 근접 사용 시 방호 조치 계획 필수 |

### 9-3. 촬영실 GUI SW (방사선 촬영 제어 소프트웨어)

| 항목 | 내용 |
|---|---|
| MFDS 등급 | 디지털의료기기로 별도 분류 (의료기기법 §2) |
| 임상시험 승인 필요 여부 | **조건부** — 독립 SW 의료기기인 경우 필요. HW와 결합 제출 시 통합 계획 가능 |
| 주요 Endpoint | 사용자 인터페이스 오류율, 촬영 파라미터 정확도, 진단 워크플로우 효율성 |
| 비임상 선행 과제 | IEC 62304 SW 검증·확인(V&V), Usability (IEC 62366-1) |
| 임상시험 특이사항 | MFDS 디지털의료기기 가이드라인 (2025.01) 기반 임상적 유효성 평가 방법 확인 필요 |

## 10. 실무 체크리스트 — MFDS 임상시험계획 승인 신청 전 확인사항

### Phase 1: 신청 전 준비

- [ ] 의료기기법 §10 면제 대상 해당 여부 확인
- [ ] MFDS 사전상담 신청 (임상시험팀, 오송)
- [ ] 비임상시험 데이터 완비 확인 (안전성 근거)
- [ ] 임상시험 실시기관 선정 (임상시험기관 지정 여부 확인)
- [ ] IRB 제출 계획 수립 (MFDS 승인과 IRB 승인 병행 가능)
- [ ] GMP 적합인정서 또는 ISO 13485 인증서 확보

### Phase 2: 서류 준비

- [ ] 임상시험계획서 작성 (KGCP·고시 2023-12호 별표 기준)
- [ ] 기술문서 (사용목적·작용원리·성능·안전성)
- [ ] Sample Size 산정 근거 (통계분석 계획서)
- [ ] Primary/Secondary Endpoint 명확 정의
- [ ] Informed Consent Form (ICF) 초안
- [ ] 이상반응 보고·모니터링 계획

### Phase 3: 신청 후 관리

- [ ] 보완 요구 시 30일 내 응답 준비
- [ ] 승인 후 CRIS (한국임상연구포털) 등록 (법적 의무)
- [ ] 중요 변경 발생 시 변경승인 선신청 후 변경 실시
- [ ] 이상반응 발생 시 MFDS 즉시 보고 절차 준수
