## 7. 주요 지역별 차별화 포인트 (Action 지시사항)

### MFDS 전용 주의사항
- **허가(신고)번호** 기재 필수 — FDA/MDR에는 없는 항목
- **의료기기 등급** (1·2·3·4등급) IFU 표시 필수
- **방사선 관계종사자** 자격 요건 명시 (방사선 발생장치)
- AI/ML 기기: **디지털의료제품법** 기반 알고리즘 설명·학습 데이터 특성·임상적 한계 필수 (2025.01~)
- 한국어 전용 작성 또는 한국어 + 외국어 병기

### FDA 전용 주의사항
- **"Rx only"** 또는 **"℞ only"** 문구 — 처방 기기 라벨·IFU 필수
- **Symbols Glossary** 별도 문서로 IFU에 첨부 또는 링크 — 심볼 사용 시 §801.15(c) 의무
- 날짜 형식: **반드시 YYYY-MM-DD** (§801.18) — 다른 형식 불가
- **Form FDA 2579**: 방사선 발생장치 연간 보고 — IFU와 별도이나 내용 일치 필요
- **Form FDA 3881** (Indications for Use): eSTAR 제출 문서와 IFU 적응증 일치 확인

### MDR 전용 주의사항
- **§23.4(z)**: 심각한 사고(Serious Incident) 발생 시 "제조사 및 회원국 관할당국에 보고" 문구 IFU에 명시 필수
- **§23.4(y)**: IFU 발행일 + 개정 식별자 (버전 번호·날짜) 병기 필수
- **§23.4(r)**: 방사선 기기는 "방사선 세기·분포" 및 "환자·사용자·주변인 보호 수단" 상세 기술 필수
- **§23.4(ab)**: SW 포함 기기 — OS 버전·HW 사양·네트워크 요건·보안 요건 구체적 기술 필수
- **다국어**: 판매 회원국별 공용어 IFU 준비 (예: 독일+프랑스+이탈리아 동시 판매 시 3개 언어)

## 8. X-ray 제품 특화 방사선 IFU 기재 요건 상세 (MDR §23.4(r) 기준)

> X-ray Detector·Handheld X-ray Source에 직접 적용. MDR §23.4(r)이 가장 상세하며 MFDS·FDA 공통 충족 기준으로 활용 가능.

| 세부 항목 | 기재 내용 예시 |
|---|---|
| 방사선 성질 | X선 (전리 방사선, 광자 에너지) |
| 방사선 유형 | 진단용 X선 (방사선학적 영상) |
| 에너지 범위 | 40~90 kVp (기기별 스펙 명시) |
| 방사선 출력 | 최대 선량률, mGy/s, 초점-피부 거리(FSD) |
| 방사선 분포 | 조사야 크기·형상 (mm × mm at 특정 거리) |
| 환자 방호 | AEC 사용·조사야 최소화·생식선 방호 권고 |
| 사용자 방호 | 0.25 mmPb 이상 방호복·갑상선 방호대 착용·최소 1.5 m 이격 |
| 주변인 방호 | 촬영 중 불필요한 인원 퇴장·이동형 사용 시 방호 병풍 |
| 비상 절차 | 오방사(Unintended Exposure) 발생 시 보고 절차 |
