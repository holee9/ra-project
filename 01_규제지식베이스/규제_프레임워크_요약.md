# 의료용 X-ray 인허가 규제 프레임워크 요약 (한눈에 보기)

> **최종 갱신**: 2026-04-22 (웹 리서치 교차검증 반영)

## 1. 국내 (MFDS)

### 1.1 법 체계 (2025년 이후 이원화)
- **전통 체계**: 의료기기법 → 시행령 → 시행규칙 → 식약처 고시·가이드라인
- **신규 체계**: 「**디지털의료제품법**」 **2025-01-24 시행** – AI·SW 중심 디지털의료제품 전용 규제
  - 변경 관리 계획(CMP) 제출 시 간단보고로 처리
  - 구성요소 성능평가 제도 (알고리즘·센서 중복 심사 방지)
  - 전주기 안전관리

### 1.2 등급 및 경로
| 등급 | 경로 | 심사 주체 |
|---|---|---|
| 1 | 신고 | 식약처 (간이) |
| 2 | 인증 | 지정 심사기관 (KTL·KTC·KTR 등) |
| 3, 4 | 허가 | 식약처 (MFDS) |

### 1.3 제출물 공통
- 기술문서 (안전성·유효성·성능)
- GMP 적합인정서 (KGMP)
- 임상시험자료 (필요 시)

### 1.4 X-ray 특화 병행 규제
- **「진단용 방사선 발생장치의 안전관리에 관한 규칙」** (의료법 제37조, 보건복지부령)
  - 최근 개정: **2025-07-18 보건복지부령 제1122호**
  - 설치 의료기관 **사용 3일 전 신고** 의무
  - **3년 주기 정기 검사**
  - 안전관리책임자 임명 필수

### 1.5 주요 디지털의료기기 가이드라인 (2024~2025)
- 디지털의료기기소프트웨어 허가·심사 가이드라인
- 독립형 디지털의료기기소프트웨어 사용적합성 허가·심사 가이드라인 (2025.01)
- 생성형 AI 의료기기 허가·심사 가이드라인 (2025.01)
- 의료기기 사이버보안 가이드라인 개정판 (2025.01)
- 디지털의료기기 표시기재 가이드라인

## 2. 미국 (FDA)

### 2.1 주요 경로
| 경로 | 대상 | 기간 (표준) |
|---|---|---|
| 510(k) Premarket Notification | Class II 대부분 | FDA review clock 90일 + 질의 대응 |
| De Novo | 저·중위해 신규 기기 | 약 150일 (FDA) |
| PMA | Class III | 약 180일 이상 |
| Pre-Submission (Q-Sub) | 사전 협의 | 70~75일 |

### 2.2 제출 방식
- **eSTAR 의무화 (2023-10-01~)**: 510(k), De Novo 대상
- RTA (Refuse to Accept) 리뷰 eSTAR에서 불필요
- 내장 양식: Truth & Accuracy statement, Form 3514, 510(k) Summary, DoC, Indications for Use (Form 3881)

### 2.3 **QMSR 시행 (2026-02-02, 이미 발효)** ⚡
- 21 CFR Part 820 개정, **ISO 13485:2016 정합 (incorporation by reference)**
- QSIT 폐기, 새 검사 프로그램 **7382.850** 사용
- FDA는 ISO 13485 인증서를 발급하지 않으며, ISO 13485 인증이 있어도 FDA 검사 면제되지 않음
- 2024년 공표 → 2년 전환기간 → **2026-02-02 시행**

### 2.4 Radiation-Emitting Product 추가 규제 (X-ray 필수)
- **21 CFR Part 1020.30 ~ 1020.33**: Diagnostic X-ray Systems Performance Standards
- Form FDA 2877 (수입 시 Initial Import Report)
- Form FDA 2579 (Report of Assembly): **2023년 개정(88 FR 3638)으로 일부 부속 컴포넌트 제출 의무 완화**
- 제조·조립·판매 이력 유지

### 2.5 Cybersecurity (법적 의무 강화)
- **FD&C Act Section 524B** (2022 Omnibus Bill 추가):
  - "Cyber Device"에 대해 **SBOM 제출 법적 의무**
  - 510(k), PMA, HDE 모두 적용
- FDA Final Guidance "Cybersecurity in Medical Devices" (**2023-09-27**)
  - Threat Model 필수
  - Cybersecurity Risk Management Report
  - SBOM (Manufacturer-developed + Third-party + Open-source)

### 2.6 AI/ML 기기
- **PCCP Final Guidance** (2024-12-03): 모든 AI-enabled device로 범위 확대 (ML 한정 아님)
- PCCP 구성: Description of Modifications + Modification Protocol + Impact Assessment
- Labeling에 "authorized PCCP 있음" 명시 요구

### 2.7 기타 필수 절차
- Establishment Registration & Device Listing (연 1회)
- UDI (Unique Device Identifier)
- MDR (Medical Device Reporting): 21 CFR 803

## 3. 유럽 (EU MDR 2017/745)

### 3.1 분류 규칙 (Annex VIII)
- **Rule 10**: 이온화 방사선을 **능동적으로 방출**하는 진단·치료용 기기 → **Class IIb**
  - 단, **"devices specifically intended for recording of diagnostic images generated by X-ray radiation" → Class IIa**
  - 즉, **X-ray Source는 IIb / X-ray Detector(수동 수신)는 IIa** 해석 가능
- **Rule 11**: SaMD (정보 제공·진단·치료 판단 영향)
  - 대부분 최소 IIa
  - 환자 immediate danger 시 IIb / 사망·중상해 가능 시 III

### 3.2 적합성 평가 절차
| 클래스 | 경로 (요약) | NB 관여 |
|---|---|---|
| I | 자기선언 | 불필요 (멸균·측정·재사용 기구 예외) |
| IIa | Annex IX/XI (일부 샘플링) | 필요 |
| IIb | Annex IX 완전 QMS + TD 심사 | 필요 |
| III | Annex IX + Design Dossier | 필요 |

### 3.3 핵심 요구사항
- GSPR (Annex I) 적합성 증명
- Technical Documentation (Annex II, III)
- QMS (ISO 13485:2016 정합)
- PMS / PMCF / PSUR / Vigilance (Eudamed)
- UDI-DI / Basic UDI-DI
- EU Authorised Representative (역외 제조사)
- PRRC (Person Responsible for Regulatory Compliance)

### 3.4 **전환기간** (Regulation **(EU) 2023/607**, 2023-03-20 발효) ⚡
Class별 전환기간 차등화:

| Class | 시한 |
|---|---|
| Class III implantable (custom-made 제외) | **2026-05-26** |
| Class III · Class IIb implantable (특정 제외) | **2027-12-31** |
| Class IIb non-implantable · IIa · Is · Im | **2028-12-31** |

**조건 (모두 충족 필요)**:
1. 기존 MDD 인증서 유효 / 계속 준수
2. 설계·용도 significant change 없음
3. 건강·안전에 unacceptable risk 없음
4. **2024-05-26까지** MDR 기반 QMS 수립
5. **2024-05-26까지** MDR NB에 공식 신청서 제출
6. **2024-09-26까지** MDR NB와 서면 계약 체결
- "Sell-off period" 삭제됨 → 시한 이전 시장 공급분은 무기한 유통 가능

### 3.5 MDCG 핵심 가이던스 (최신)
- **MDCG 2019-11 Rev.1** (2025-06): SW 정의·분류 – AI 기반 X-ray 영상 평가 SW는 항상 MDSW로 분류
- MDCG 2019-16: Cybersecurity
- MDCG 2020-5/6/7/8: Clinical Evaluation / Legacy / PMCF
- MDCG 2022-14: MDR 전환
