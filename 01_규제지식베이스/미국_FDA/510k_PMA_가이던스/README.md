# 미국 FDA - 510(k) / PMA 가이던스

## 수록 대상
- Premarket Notification 510(k) 제출 가이던스
- PMA (Premarket Approval) 관련 문서
- De Novo Classification Request 관련 문서
- eSTAR / eSubmitter 제출 양식 및 매뉴얼

## X-ray 관련 핵심 가이던스
- "Pediatric X-ray Imaging" Guidance
- "Solid State X-ray Imaging Devices" Guidance (FDA)
- "Radiation Safety Considerations for X-ray Equipment"
- "Software as a Medical Device (SaMD)" Guidance
- "Cybersecurity in Medical Devices" Guidance (2023)
- "Clinical Performance Assessment: Considerations for CAD Devices"

## 제출 절차 개요
1. Predicate Device 조사 (FDA 510(k) Database)
2. Substantial Equivalence 비교표 작성
3. 510(k) Summary, 성능시험, 생체적합성, 전자파 등 섹션 준비
4. eSTAR 템플릿을 통한 전자 제출
5. FDA 심사 질의(AI: Additional Information) 대응
