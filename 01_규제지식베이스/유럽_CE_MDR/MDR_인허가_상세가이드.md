# EU MDR 인허가 상세 가이드 (X-ray System 전용)

> 웹 리서치 교차검증 반영 – 2026-04-22 기준

## 1. 핵심 법 체계

### 1.1 Regulation
- **Regulation (EU) 2017/745 (MDR)**: 2021-05-26 시행
- **Regulation (EU) 2023/607**: 2023-03-20 발효 – **전환기간 연장**
- 관련: Regulation (EU) 2017/746 (IVDR)

### 1.2 부속서 (Annex) 핵심
- Annex I: GSPR (General Safety and Performance Requirements)
- Annex II: Technical Documentation
- Annex III: PMS (Post-Market Surveillance) TD
- Annex VII: NB 요구사항
- Annex VIII: **분류 규칙 (Rule 1~22)**
- Annex IX/X/XI: 적합성 평가 절차
- Annex XV: Clinical Investigation

## 2. 분류 규칙 (Annex VIII) – X-ray 관련

### 2.1 Rule 10 (핵심)
> "Active devices intended to emit ionising radiation and intended for diagnostic or therapeutic radiology, including interventional radiology devices and devices which control or monitor such devices, or which directly influence their performance, shall be classified as class IIb."

**→ Handheld X-ray Source = Class IIb**

### 2.2 Rule 10 후단
> "However, devices specifically intended for recording of diagnostic images generated by X-ray radiation shall be classified as class IIa."

**→ X-ray Detector (수동 영상 수신·기록 기기) = Class IIa**

### 2.3 Rule 11 (SW)
> Software intended to provide information which is used to take decisions with diagnosis or therapeutic purposes is classified as class IIa.
> 단, decision may cause (a) death or irreversible deterioration → **Class III**; (b) serious deterioration or surgical intervention → **Class IIb**.
> Software intended to monitor physiological processes: 일반 IIa / immediate danger IIb.
> Others: Class I.

**→ 촬영실 GUI SW = Rule 11 적용, 영향도에 따라 IIa ~ III**

## 3. 적합성 평가 경로

| Class | 경로 (예) | NB | 문서 요구 |
|---|---|---|---|
| I | Annex I + 자기선언 | 불필요 (일부 예외) | TD + DoC |
| Is, Im, Ir | Annex IX (Ch.I) 또는 Annex XI | 필요 (해당 요소만) | + QMS |
| IIa | Annex IX (Ch.I + III, 샘플링) | 필요 | TD 심사 샘플링 |
| IIb | Annex IX (Ch.I + III, 전수) | 필요 | 전수 심사 |
| III | Annex IX + Ch.II (Design Dossier) 또는 Annex X+XI | 필요 | Design Dossier |

**자사 제품 대응**:
- Handheld X-ray Source (IIb) → Annex IX Ch.I + III
- X-ray Detector (IIa) → Annex IX (샘플링) 또는 Annex XI
- GUI SW (IIa/IIb) → 영향도 분석 후 결정

## 4. GSPR (Annex I) 대응

### 4.1 Chapter I – General Requirements
- 안전·성능 원칙 (§1~9)
- 위험관리 (ISO 14971 조화)
- Benefit-Risk 평가

### 4.2 Chapter II – Performance/Design/Manufacture
- 화학·물리·생물 속성
- 감염·미생물 오염
- 방사선 (§16)
- 에너지·의도 에너지원 (§17)
- 진단·측정 (§15)

### 4.3 Chapter III – Information (라벨·IFU)
- 라벨·IFU 최소 요건 (§23)
- 심볼: ISO 15223-1:2021

**→ GSPR 체크리스트는 각 제품 기술파일에 필수 포함**

## 5. Technical Documentation (Annex II, III)

### 5.1 Annex II (기기 TD)
1. 기기 설명·사양
2. 제조자 정보
3. 설계·제조 정보
4. GSPR
5. Benefit-Risk + Risk Management
6. 제품 검증·유효화
7. PMS

### 5.2 Annex III (PMS TD)
- PMS Plan
- PSUR (Periodic Safety Update Report)
- PMCF Plan (IIa 이상 필수 원칙)
- PMCF Evaluation Report

## 6. PMS · PMCF · PSUR · Vigilance

| 항목 | 주기 | 제출처 |
|---|---|---|
| PMS Plan | 지속 운영 | 내부 + NB 심사 시 |
| PSUR | Class IIa: ≥2년마다 / IIb·III: 연 1회 | Eudamed / NB |
| PMCF Evaluation Report | PMS 일부로 갱신 | 내부 |
| Vigilance (Serious Incident) | 15일 / 2일 (심각) | Competent Authority + Eudamed |
| Field Safety Corrective Action | 즉시 | CA + Eudamed |
| Trend Report | 유의미 경향 탐지 시 | CA |

## 7. 전환기간 (Regulation (EU) 2023/607)

### 7.1 차등 시한
| Class | 시한 |
|---|---|
| Class III **custom-made** implantable | 2026-05-26 |
| Class III / Class IIb **implantable** (sutures·screws 등 제외) | **2027-12-31** |
| Class IIb **non-implantable** / IIa / Is / Im | **2028-12-31** |

### 7.2 조건 (전부 충족 필요)
1. AIMDD/MDD 인증서 유효·준수 지속
2. 설계·intended purpose **significant change 없음**
3. 건강·안전 unacceptable risk 없음
4. **2024-05-26까지** MDR QMS 수립
5. **2024-05-26까지** MDR NB에 공식 신청서 제출
6. **2024-09-26까지** NB와 **서면 계약** 체결
- "Sell-off period" 삭제 → 시한 이전 투입분 무기한 유통 가능

## 9. 주요 역할 (역외 제조사 필수)

### 9.1 EU Authorised Representative
- EU 내 소재 필수
- 책임: TD 보관, CA·NB 대응, Vigilance 접수

### 9.2 PRRC (Person Responsible for Regulatory Compliance)
- 제조자 내부 지정
- 자격 요건 (학위+경력 또는 경력) MDR Article 15
- 역할: 제품 release 감독, PMS, Vigilance 조정

### 9.3 Importer / Distributor
- 의무: 라벨·UDI 확인, 보관·운송 조건 유지, 불만·Vigilance 전달

## 10. UDI 및 Eudamed
- **Basic UDI-DI**: 제품군 단위
- **UDI-DI**: 모델 단위
- Eudamed 모듈:
  - Actor Registration
  - UDI/Device Registration
  - NB / Certificates
  - Clinical Investigation
  - Vigilance
  - Market Surveillance
- 현재 일부 모듈만 의무 – 공식 공고에 따라 단계적 확장
