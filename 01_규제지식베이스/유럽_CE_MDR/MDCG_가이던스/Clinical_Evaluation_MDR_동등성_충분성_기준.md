# Clinical Evaluation (MDR) 심화 — 동등성·충분성 기준

## 1. 개요: MDR Article 61 구조

MDR Article 61은 임상평가(Clinical Evaluation)의 **의무**와 **임상조사(Clinical Investigation) 면제 조건**을 규정한다.

```
Article 61 핵심 구조
────────────────────────────────────────────────
(1) 모든 기기: 임상평가 수행 의무 (GSPR 적합성 확인)
(3) Class I/IIa/IIb 비이식형: 충분한 임상데이터로 임상조사 면제 가능
(4) 이식형 + Class III: 임상조사 원칙 의무
    └ 예외 CASE 1: 동일 제조사 변형 기기 + 동등성 입증 + NB 승인
(5) 타사 기기 동등성 주장: 계약 필요 (→ CASE 4)
(6) Legacy device 특례:
    └ (a) MDD/AIMDD 적법 시판 기기 + 충분한 임상데이터 → CASE 2
    └ (b) 봉합사·스테이플 등 열거 기기 → CASE 3
(10) 예외적 임상데이터 없이 적합성 입증 가능 (비이식형·비Class III에 한정)
(11) 임상평가는 제품 수명 전 기간 지속 갱신 의무
────────────────────────────────────────────────
```

**핵심 용어 정의 (MDR Article 2)**

| 용어 | 정의 |
|------|------|
| Clinical data (임상데이터) | 기기 또는 동등 기기의 사용에 관한 안전·성능 정보 (Art. 2(48)) |
| Clinical evidence (임상 근거) | 임상데이터의 양·질 평가 결과 (Art. 2(51)) |
| Clinical performance (임상 성능) | 기기가 의도한 목적 달성 능력, 측정 가능한 임상적 효과 (Art. 2(52)) |
| Clinical benefit (임상적 이익) | 환자에 대한 긍정적 효과 — 진단·예방·치료 등 (Art. 2(53)) |
| Equivalent device (동등 기기) | 동등성 입증된 이미 시판 중인 기기 (Annex XIV §3) |

## 2. 동등성(Equivalence) — 3대 기준 (Annex XIV Part A §3 + MDCG 2020-5)

동등성 주장을 위해서는 **3가지 특성 전체**를 동시에 충족해야 한다.  
MEDDEV 2.7/1 rev. 4 대비 MDR의 강화 요건을 아래에 명시한다.

### 2.1 기술적 특성 (Technical Characteristics)

| MDR 요건 | MEDDEV 2.7/1 rev.4 대비 차이 |
|----------|------------------------------|
| 유사한 설계(similar design) | 동일 기준 |
| **유사한** 사용 조건(similar conditions of use) | MEDDEV는 **동일**(same) — MDR이 완화 |
| 유사한 사양·특성 (물리화학적 특성, 강도, 점도, 표면, 파장) | 동일 기준 |
| **소프트웨어 알고리즘 포함** (MDR 명시 추가) | MEDDEV 미명시 — MDR 강화 |
| 유사한 적용 방법 (해당 시) | 동일 기준 |
| 유사한 작동 원리 및 핵심 성능 요건 | 동일 기준 |

> **SW 알고리즘 동등성 판단 기준** (MDCG 2020-5 §3.1(b)):  
> - SW 코드 자체의 동일성이 아닌, **기능적 원리·임상 성능·의도된 목적**이 유사한지 판단  
> - 단, SW가 국제표준(IEC 62304, IEC 82304-1)에 따라 개발·검증된 경우 코드 레벨 동등성 요구 불필요  
> - 기기 설정 UI(표시 방식 등) 전용 SW는 의료 목적에 영향 없으면 동등성 요건 적용 제외

### 2.2 생물학적 특성 (Biological Characteristics)

| MDR 요건 | MEDDEV 2.7/1 rev.4 대비 차이 |
|----------|------------------------------|
| 동일 조직·체액과 접촉하는 **동일한** 재료·물질 사용 | **MDR 강화**: MEDDEV의 '유사 재료' 예외 조항 **인정 안 됨** |
| 유사한 접촉 종류 및 기간 | 동일 기준 |
| 유사한 물질 방출 특성 (분해산물, 용출물 포함) | MDR 명시 추가 — 분해산물·용출물 명시 |

> **주의**: MDR은 재료의 "same materials or substances" 요건에서 예외를 허용하지 않는다.  
> ISO 10993-18(화학적 특성화), ISO 10993-17(독성 위험평가) 적용으로 최종 기기 단위 평가 필요.

### 2.3 임상적 특성 (Clinical Characteristics)

| MDR 요건 | MEDDEV 2.7/1 rev.4 대비 차이 |
|----------|------------------------------|
| 동일 임상 상태·목적 (유사한 중증도·병기 포함) | 동일 기준 |
| 신체 동일 부위 (**"same"** 표현) | 동일 기준 |
| 유사한 환자군 (연령·해부학·생리학 포함) | 동일 기준 |
| **동일한 사용자 유형** (same kind of user) | **MDR 추가**: MEDDEV 미명시 |
| 의도된 목적 대비 유사한 핵심 임상 성능 | 동일 기준 |

> **사용자 유형 판단**: 전문 의료인(HCP) 대상 기기 vs. 가정용(lay person) 기기는  
> 동일 임상 상태라도 동등 기기로 인정 불가.

### 2.4 동등성 입증 전제 조건

```
동등성 입증 필수 조건 체크
 ① 3가지 특성 모두 충족 ("similar to the extent that there would be no 
    clinically significant difference in safety and clinical performance")
 ② 하나의 동등 기기는 전체 특성에서 동등해야 함 — 여러 기기의 부분 조합 불가
    (예외: 동일 제조사의 기기 시스템에서 부분 시스템 동등성 주장 가능)
 ③ 동등 기기 데이터에 대한 "sufficient levels of access" 입증 필요
    (Annex XIV §3 마지막 문단)
 ④ 동등 기기 데이터는 특정 세대·버전 단위로 관리 필요
 ⑤ Annex XVI 제품(의료 목적 없는 제품)과는 동등성 주장 불가
```

**데이터 접근 수준 계층 (MDCG 2023-7 Appendix II)**

| 접근 수준 | 방법 | 계약 필요 |
|-----------|------|-----------|
| 최상 (Full) | 동일 제조사 변형 기기 / 계약 체결 | Art. 61(5): 필요 |
| 상 (High) | 동일 생산 하청업체 공급 기기 + 공개 임상데이터 | 불필요 |
| 중 (Medium) | 샘플 비교분석 + 공개 정보 | 불필요 |
| 하 (Low) | 공개 정보만 | 불필요 (단, 한계 보완 필요) |

## 3. 충분성(Sufficiency) — 임상 근거의 충분성 판단 (MDCG 2020-6)

### 3.1 충분한 임상 근거의 정의

> "Sufficient clinical evidence" = 임상평가 결과, **관련 GSPR(일반안전성·성능요건) 적합성**이 확인되는 수준의 임상데이터의 양·질을 갖춘 상태  
> — MDCG 2020-6, §1.2

충분성은 다음 요소를 종합하여 판단한다:

| 판단 요소 | 세부 내용 |
|-----------|-----------|
| **임상데이터 양** | 연구 수, 환자 수, 추적 기간 |
| **임상데이터 질** | 연구 설계(RCT > 코호트 > 케이스 시리즈), 비뚤림 위험 |
| **데이터 관련성** | 기기의 의도된 목적, 대상 환자군, 사용 환경과의 일치도 |
| **안전성 데이터** | 이상사례 발생률, 중증 이상사례 유형·빈도 |
| **임상적 이익** | 측정 가능한 임상 결과지표(endpoint)로 입증된 효과 |

### 3.2 Legacy Device 충분성 (MDCG 2020-6 §4)

MDD/AIMDD 하에서 이미 CE 인증을 받은 기기(레거시 기기)가 MDR 재인증 시 충분성 입증 경로:

```
레거시 기기 임상데이터 소스 (Article 61(6)(a) 적용 시)
─────────────────────────────────────────────────────
① PMCF 데이터 (MDD 시절 수집)
② PMS 안전 데이터 (판매 후 감시)
③ 레지스트리 데이터 소급 연구
④ 과학 문헌 기반 독립 연구 데이터
⑤ 이전 임상조사 결과 (MDD 기준 수행)
─────────────────────────────────────────────────────
주의: MDR 발효 이후 신규 기기는 레거시 경로 적용 불가
```

### 3.3 임상조사 면제 4가지 Case (MDCG 2023-7)

이식형·Class III 기기에만 적용 (비이식형·Class I/IIa/IIb는 별도 판단).

| Case | 근거 조항 | 핵심 조건 | 계약 필요 |
|------|-----------|-----------|-----------|
| **CASE 1** | Art. 61(4) indent 1-3 | ① 동일 제조사 변형 기기 ② 동등성 입증 + NB 승인 ③ 기존 CE 임상평가 충분 ④ PMCF 계획 적절 | **불필요** |
| **CASE 2** | Art. 61(6)(a) | ① MDD/AIMDD 적법 시판 레거시 기기 ② 충분한 임상데이터 ③ 관련 CS(Common Specification) 준수 | **불필요** |
| **CASE 3** | Art. 61(6)(b) | ① 봉합사·스테이플·치과충전재 등 열거 기기(WET) ② 충분한 임상데이터 ③ 관련 CS 준수 | **불필요** |
| **CASE 4** | Art. 61(5) | ① CASE 1 조건 충족 ② 타사 기기 대상 ③ 양사 간 기술문서 접근 계약 ④ 동등 기기 임상평가가 MDR 기준 수행 | **필수** |

> **중요**: CASE 1~3은 상호 독립적이며 Art. 61(5) 계약 요건이 교차 적용되지 않는다.  
> — MDCG 2023-7 §4

## 4. FDA 비교 — Substantial Equivalence (실질적 동등성)

### 4.1 510(k) 실질적 동등성 판단 기준

```
SE(Substantial Equivalence) 판단 흐름
──────────────────────────────────────────
① 의도된 용도(Intended Use) 동일?
   YES → ③
   NO  → De Novo 또는 PMA 고려

② 기술적 특성(Technological Characteristics) 동일?
   YES → SE 판정
   NO  → ③

③ 기술적 특성 차이가 새로운 안전·유효성 문제 야기하는가?
   YES → PMA 고려
   NO  → 비임상/임상 데이터로 차이 입증 가능?
         YES → SE 판정 (데이터 제출)
         NO  → Not SE
──────────────────────────────────────────
```

### 4.2 임상데이터 필요 시점 (FDA 2023년 Draft Guidance)

- 의도된 용도 차이 존재 시
- 기술적 특성 차이가 안전·유효성 문제 야기 가능 시
- 비임상 시험으로 동등성 입증 불충분 시
- Predicate 기기에 새로운 위험 발견 시

### 4.3 MDR Equivalence vs. FDA Substantial Equivalence 비교

| 비교 항목 | MDR 동등성 | FDA SE |
|-----------|-----------|--------|
| 근거 | Annex XIV §3 + Article 61 | 21 USC §360c(i) + 510(k) Program Guidance |
| 3대 특성 | 기술·생물·임상 전체 충족 | 주로 의도된 용도 + 기술적 특성 |
| 생물학적 특성 | 동일 재료 요건 (엄격) | 별도 Biocompatibility(ISO 10993) 요건으로 관리 |
| 임상데이터 | 원칙 필요 (Art. 61(1)) | 비임상으로 충분한 경우 면제 가능 |
| 소프트웨어 | SW 알고리즘 동등성 명시 | SW Documentation + 기능 비교 |
| 역사적 데이터 | Legacy 경로(MDCG 2020-6) | Pre-amendment device 또는 기존 SE 기기 |
| 타사 기기 접근 | 공개 문헌 활용 가능 (계약 없이) | Predicate 공개 510(k) 데이터 활용 |

## 5. MFDS 비교 — 국내 동등성·임상시험 면제

### 5.1 MFDS 동등성 비교 기준 (고시 제2024-88호)

MFDS는 이미 허가된 기기와의 동등성 비교를 통해 임상시험 자료 제출을 면제할 수 있으며,  
동등성 판단 기준은 다음과 같다.

| 비교 항목 | 내용 |
|-----------|------|
| 사용목적 | 동일 또는 유사 |
| 작용원리 | 동일 또는 유사 |
| 구조·재료 | 동일 또는 유사 |
| 성능 | 동일 또는 유사 |
| 안전성 | 동일 수준 이상 |

### 5.2 임상시험 자료 제출 원칙

- **신개발의료기기**: 임상시험 자료 원칙 필요
- **기존 허가 기기 동등성 비교 가능 시**: 임상시험 자료 면제 가능 (단, 시판 후 조사기간 중 동등성 면제 제한 있음)
- **3등급 X-ray 기기**: 임상 성능 시험 자료 또는 동등성 비교 자료 필요

### 5.3 MFDS CER 관련 요건

MFDS는 독립적인 Clinical Evaluation Report(CER) 서식을 별도 지정하지 않으나,  
기술문서 내 임상 성능 관련 자료로 다음을 요구:
- 임상문헌 조사 결과
- 임상시험 성적서 (해당 시)
- 동등 기기 임상 자료 (동등성 비교 경로 선택 시)

## 6. 3지역 임상평가 비교 매트릭스

| 항목 | MFDS (한국) | FDA (미국) | MDR (EU) |
|------|------------|-----------|---------|
| **근거 법령** | 고시 제2024-88호 §13~§15 | 21 USC §360c(i); 21 CFR §807 | MDR Art. 61 + Annex XIV |
| **주요 가이던스** | MFDS 의료기기 CER 작성 가이드라인 | 510(k) Program Guidance (2014) | MDCG 2020-1/5/6, 2023-7, 2024-3 |
| **동등성 기준** | 사용목적·작용원리·구조·성능·안전성 | Intended use + Technical char. | Technical + Biological + Clinical |
| **생물학적 요건** | ISO 10993 시험으로 별도 관리 | ISO 10993 시험으로 별도 관리 | **동일 재료** 요건 (동등성 입증 내 포함) |
| **SW 동등성** | 기능·성능 동일성 위주 | SW Documentation 비교 | **알고리즘** 명시 요건 |
| **임상조사 면제** | 동등성 비교 자료 또는 문헌 검토 | 비임상 벤치 데이터로 가능 | 4가지 Case 구조 (MDCG 2023-7) |
| **이식형·Class III 기기** | 임상시험 자료 원칙 필요 | PMA: 임상조사 의무 | Art. 61(4) 임상조사 원칙 의무 |
| **임상평가 지속 갱신** | 허가 후 변경 시 갱신 | MDR report 별도 없음 | Art. 61(11): 전 수명주기 갱신 의무 |
| **PMCF/PMS** | 시판 후 조사 | MDR 별도 없음 | PMCF Plan + Report 의무 |
| **NB/제3자 검토** | 해당 없음 (심사기관) | FDA 자체 심사 | Implantable/Class III: NB 필수 검토 |

## 7. 3개 제품 영향 맵핑

### 7.1 X-ray Detector

| 규제 항목 | 세부 내용 |
|-----------|-----------|
| **MDR 분류** | Class IIa (Rule 10 — 방사선 이미징) |
| **임상조사 의무** | 없음 — 충분한 임상데이터 + 동등성으로 대체 가능 |
| **동등성 핵심 포인트** | ① 기술: DQE·MTF·NPS 특성·픽셀 피치·에너지 범위 유사성 ② 생물: 환자 피부 미접촉 기기 — 생물학적 특성 요건 완화 가능 ③ 임상: 동일 X-ray 촬영 목적·동일 의료진 사용자 |
| **FDA** | 510(k) — 기존 flat-panel detector 대비 SE 입증 (비임상 벤치 데이터로 충분할 가능성 높음) |
| **MFDS** | 3등급 의료기기 — 동등 기기 임상 문헌 검토로 임상시험 면제 가능 |
| **CER 핵심 섹션** | DQE 성능 데이터, 방사선 선량 비교, 동등 Detector 임상 문헌 |
| **PMCF** | 필요 — 신제품 출시 후 실사용 안전성 모니터링 |

### 7.2 Handheld X-ray Source

| 규제 항목 | 세부 내용 |
|-----------|-----------|
| **MDR 분류** | Class IIb (Rule 9 — 방사선 발생 기기) |
| **임상조사 의무** | 없음 (Class IIb) — 그러나 충분성 입증 요건 강화 |
| **동등성 핵심 포인트** | ① 기술: 관전압(kV)·관전류(mA)·선량 특성·초점 크기·SID(선원-피부거리) 유사성 ② 생물: 비접촉 기기 — 생물학적 특성 N/A ③ 임상: 동일 촬영 부위 목적·동일 사용자 유형(방사선사/의사) |
| **FDA** | 510(k) — 기존 handheld portable X-ray 대비 SE, 방사선 선량 차이 시 임상데이터 필요 가능성 있음 |
| **MFDS** | 3등급 방사선 발생기 — 동등 기기 임상 자료 제출 필요. 신규 형태(handheld)는 임상시험 자료 요구 가능 [firm 임상전략] |
| **CER 핵심 섹션** | 방사선 선량 평가 (IEC 60601-1-3), 사용 환경 안전성 (산란방사선), 임상 문헌 |
| **PMCF** | 필요 — handheld 폼팩터 특성상 신규 사용 패턴 모니터링 필요 |

### 7.3 촬영실 GUI SW

| 규제 항목 | 세부 내용 |
|-----------|-----------|
| **MDR 분류** | Class IIa (MDCG 2019-11 기준, 진단 지원 SW); Class IIb 가능 (AI/진단 결과 영향 시) |
| **임상조사 의무** | 없음 — SW 동등성 + 임상 문헌으로 대체 가능 |
| **동등성 핵심 포인트** | ① 기술: **SW 알고리즘 동등성이 가장 중요** — 이미지 처리·워크플로우·출력 포맷 유사성 ② 생물: N/A (SW는 직접 환자 접촉 없음) ③ 임상: 동일 촬영실 워크플로우·동일 사용자(방사선사·의사) |
| **FDA** | 510(k) + SW Documentation (IEC 62304 기반) — 알고리즘 기능 비교표 필요 |
| **MFDS** | 독립형 SW 2~3등급 — 동등 SW 임상 문헌 또는 성능 검증 자료 |
| **CER 핵심 섹션** | SW 알고리즘 기능 검증 결과, 사용성 평가(IEC 62366), 임상 워크플로우 성능 데이터 |
| **PMCF** | 필요 — SW 업데이트 주기 고려, 사용 중 이상 보고 추적 |

## 8. 동등성 입증 테이블 작성 가이드

MDCG 2020-5 Annex I 기반 Equivalence Table 필수 구조:

```
[기술적 특성]
1.1 유사한 설계          → 기기A 기술 문서 참조 | 기기B(동등 기기) 설명 | 차이 식별
1.2 유사한 사용 조건     → 동일 사용 환경 입증
1.3 유사한 사양·특성     → SW 알고리즘 포함 상세 비교
1.4 유사한 적용 방법
1.5 유사한 작동 원리     → 임상적 유의적 차이 없음 과학적 근거

[생물학적 특성]
2.1 동일한 접촉 재료·물질 → "same"이어야 함 (유사 불가)
2.2 유사한 접촉 종류·기간
2.3 유사한 방출 특성

[임상적 특성]
3.1 동일한 임상 상태·목적
3.2 동일한 신체 부위
3.3 유사한 환자군
3.4 동일한 사용자 유형    → HCP vs. lay person 구분 필수
3.5 유사한 핵심 임상 성능

[종합 결론]
→ 개별 차이의 합산 효과(additive effect)가 임상적으로 유의한 차이 야기하지 않음을 입증
```

## 9. 실무 제출 체크리스트

### 9.1 MDR CER 동등성 섹션 체크리스트

- [ ] Equivalent Device 선정 근거 명시 (동일 GSPT 내 최유사 기기)
- [ ] Equivalence Table (기술/생물/임상 3대 특성 전항목) 작성
- [ ] 각 차이에 대한 과학적 정당화 근거 첨부
- [ ] "sufficient levels of access" 입증 문서 (계약서 또는 공개 데이터 접근 근거)
- [ ] SW 알고리즘 동등성 분석 (MDCG 2020-5 §3.1(b) 기준)
- [ ] 이식형·Class III 기기의 경우 면제 Case 명시 (CASE 1~4 중 해당)
- [ ] 생물학적 특성: "same materials" 입증 (ISO 10993 계열 데이터 연계)

### 9.2 충분성(Sufficiency) 평가 체크리스트

- [ ] 임상데이터 소스 목록 작성 (문헌·임상조사·PMS 데이터 등)
- [ ] 각 데이터 소스 품질 평가 (연구 설계, 비뚤림 위험, 관련성)
- [ ] 안전성 데이터: 이상사례·심각한 이상사례 발생률 정리
- [ ] 임상 성능 데이터: 의도된 목적 대비 측정된 임상적 이익 입증
- [ ] Residual risk 식별 및 위험-이익 분석 (ISO 14971 연계)
- [ ] 충분성 결론 — GSPR 적합성 확인 가능 수준인지 명시
- [ ] PMCF Plan 작성 (MDCG 2020-7 기준)

### 9.3 FDA 510(k) SE 체크리스트

- [ ] Predicate 기기 선정 근거 (가장 유사한 기기, 최신 510(k) 우선)
- [ ] Intended Use 동일성 확인
- [ ] 기술적 특성 차이 식별 및 안전·유효성 영향 평가
- [ ] 비임상 성능 데이터 (벤치 시험)
- [ ] 임상데이터 필요 여부 판단 (FDA 2023 Draft Guidance 기준)
- [ ] SW 기기의 경우 알고리즘 기능 비교표 포함

### 9.4 MFDS 체크리스트

- [ ] 기 허가 동등 기기 식별 (식약처 의료기기 제품정보방)
- [ ] 동등성 비교표 작성 (사용목적·작용원리·구조·성능·안전성)
- [ ] 임상 문헌 검토 자료 (국내외 논문, 3편 이상 권장)
- [ ] 임상시험 면제 근거 명시 (동등성 경로 또는 문헌 경로)
- [ ] 신개발의료기기 해당 시 임상시험 계획 수립
