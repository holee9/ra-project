## 1. 적용 대상 제품 판단

### 1.1 본 가이던스 적용 범위
- 전자 프로그래밍 시스템(Electronic Programmable Systems)을 포함하는 기기
- 그 자체가 의료기기인 소프트웨어(MDSW)
- MDR/IVDR 적용 대상 모든 의료기기

### 1.2 자사 3개 제품 적용 판단

| 제품 | MDCG 2019-16 적용 | 핵심 이유 |
|---|---|---|
| X-ray Detector | ✅ 적용 | 디지털 센서·소프트웨어 포함, 네트워크 연결 가능 |
| Handheld X-ray Source | ✅ 적용 | 전자 제어 시스템·소프트웨어 포함 |
| 촬영실 GUI SW | ✅ 적용 (주요 대상) | MDSW 직접 해당, IEC 62304 Class B/C |

## 6. 이해관계자별 역할 분담

| 역할 | 핵심 책임 |
|---|---|
| **제조사** | 보안 설계, 위험 관리, IFU 정보 제공, 패치 발행, PMS, 인시던트 보고 |
| **통합자(Integrator)** | 운영환경 보안 수준 평가, 보안 구성 설정, 패치 지원, 문서·교육 제공 |
| **운영자(Operator/HDO)** | 운영환경 보안 유지, 인프라 제공, 패치 적용, 이상 시 제조사 즉시 통보 |
| **사용자(HCP/환자)** | IFU 준수, 최신 소프트웨어 유지, 강력한 비밀번호 사용, 의심 활동 보고 |

> **핵심 원칙**: 의료기기 무단 변경(SW 패치, 방화벽 변경, 타사 SW 설치 등)은 **반드시 제조사 공개 지침** 하에만 수행. 무허가 변경은 안전에 예측 불가능한 영향 초래 가능.
