# ISO 15223-1:2021 심볼 적용 가이드

## 1. 표준 개요

| 항목 | 내용 |
|---|---|
| 표준 번호 | ISO 15223-1:2021 (EN ISO 15223-1:2021) |
| 전체 제목 | Medical devices — Symbols to be used with information to be supplied by the manufacturer — Part 1: General requirements |
| 발행일 | 2021-11 |
| 대체 | ISO 15223-1:2016 |
| 최신 Amendment | ISO 15223-1:2021/Amd 1:2025 (2025-03-05 발행) |
| Amendment 내용 | EC REP 심볼 → [XX]-REP 형식 변경 (EU-REP 포함), Authorized Representative 정의 추가 |

**2016 → 2021 주요 변경점:**
- 표준 제목 변경: "symbols to be used with medical device labels" → "symbols to be used with information to be supplied by the manufacturer" (범위 확장)
- 의료기기 식별 심볼 (MD 심볼) 신규 추가: MDR Annex I §23.2(q) 대응
- 일부 심볼 정의 명확화 및 사용 사양 개정

**2025 Amendment 핵심:**
- EC REP 심볼: `[EC]` → `[XX]` (국가/지역 코드 자리)
- EU 시장: `EU-REP`, 스위스: `CH-REP` 등 관할권별 적용
- 전환 타임라인 미확정 (2026-05 기준). 기존 EC REP 사용 라벨 즉시 변경 의무 없음

## 2. 3개 지역 적용 요건 비교

| 요소 | MFDS (한국) | FDA (미국) | MDR (EU) |
|---|---|---|---|
| **참조 규정** | 의료기기 표시·기재 등에 관한 규정 (2024-12-04 시행) | 21 CFR Part 801, § 801.15 | MDR (EU) 2017/745 Annex I §23 |
| **ISO 15223-1 인정 여부** | 인정 (국제규격 준용) | FDA 공인 합의 표준 (Recognition #42873) | 조화 표준 (OJEU 게재) |
| **심볼 단독 사용** | 원칙적 한국어 병기 필요; 국제 표준 심볼 사용 인정 시 예외 가능 | FDA 공인 표준 심볼: adjacent text 불요. Symbols Glossary 의무 작성 | ISO 15223-1 심볼 사용 권장; 사용 언어는 해당 회원국 언어 요건 적용 |

| **MD 식별 심볼** | 요구 없음 | 요구 없음 | MDR Annex I §23.2(q): SaMD 및 기기 식별용 MD 심볼 사용 권장 |
| **EU-REP 심볼** | 해당 없음 | 해당 없음 | Annex I §23.2(f): Authorized Representative 표시 의무 (Amd.1:2025 후 EU-REP) |

### FDA Symbols Glossary 세부 요건 (21 CFR 801.15)
심볼을 adjacent text 없이 사용할 경우 반드시 Symbols Glossary 작성 (라벨 또는 IFU 내 수록):
1. 사용한 각 심볼
2. 표준명 및 표준 번호 (예: ISO 15223-1:2021)
3. 심볼 제목 및 표준 내 참조 번호
4. FDA 인정 시 명시된 의미/설명 텍스트

## 5. 라벨 설계 시 주의 사항

### 5.1 MFDS (한국)
- 국문 표시 원칙: 심볼 사용 시 한국어 설명 병기 권고 (ISO 심볼 단독 인정 여부는 품목·담당자 확인)
- 의료기기 표시·기재 등에 관한 규정 (2024-12-04 시행) 준수
- 수입 기기: 수입원 정보 추가 필요
- UDI: UDI 포털 등록 후 라벨에 DI (Device Identifier) 표시

### 5.2 FDA (미국)
- ISO 15223-1:2021 FDA 공인 확인 → Symbols Glossary 작성 (IFU 내 또는 별도 페이지)
- eSTAR Form FDA 3881: 라벨 심볼 목록 → Symbols Glossary 첨부
- 표준 외 심볼 사용 시: adjacent explanatory text 必須
- Rx 기기: "Caution: Federal law restricts this device to sale by or on the order of a physician." 또는 Rx 심볼 + 설명

### 5.3 MDR (EU)
- Harmonized standard EN ISO 15223-1:2021 준수 → 적합 추정 (presumption of conformity)
- EU-REP: ISO 15223-1:2021/Amd 1:2025 후 `EU-REP` 사용 (기존 `EC REP` 전환 타임라인 모니터링)
- MD 심볼 (§5.5.1): MDR Annex I §23.2(q) 대응 — 기기가 의료기기임을 명시
- 현지 언어 요건: 판매 회원국 공식 언어로 IFU 제공 (심볼은 언어 중립적으로 사용 가능)
- Technical Documentation: 사용한 모든 심볼과 표준 참조 번호를 기술문서에 목록화

## 6. 심볼 구현 체크리스트

### 설계 단계
- [ ] 적용 시장(MFDS/FDA/MDR) 확정 → 각 지역 필수 심볼 목록 작성
- [ ] ISO 15223-1:2021 최신본 및 Amd 1:2025 검토
- [ ] FDA 사용 시: Symbols Glossary 작성 및 IFU에 수록
- [ ] EU 수출 시: EU-REP 심볼 적용 (Amd 1:2025 전환 타임라인 확인)
- [ ] UDI 바코드 (GS1 DataMatrix 또는 HRI) 포함 여부 확인

### 검증 단계
- [ ] 심볼 크기·대비 → IEC 60601-1-6 / ISO 9186-1 가독성 기준 확인
- [ ] 라벨 표면 내구성 시험 (ISO 15223-1 §4 요건: 정상 사용 조건 유지)
- [ ] 3개 지역 심볼 사용 사양(specification for use) 준수 여부 교차 확인
- [ ] Usability Testing: 대표 사용자 대상 심볼 이해도 확인 (IEC 62366-1 연계)

### 제출 단계
- [ ] MFDS: 기술문서 내 라벨 샘플 및 심볼 목록 제출
- [ ] FDA (510k): eSTAR 라벨 섹션 + Symbols Glossary 첨부
- [ ] MDR: Technical Documentation Annex II §6.1 라벨 및 IFU 섹션에 수록
