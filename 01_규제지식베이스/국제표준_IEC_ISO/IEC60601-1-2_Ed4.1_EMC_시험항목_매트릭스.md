# IEC 60601-1-2:2014+A1:2020 (Ed. 4.1) EMC 시험 항목·Pass Criteria 매트릭스

## 1. 표준 개요

| 항목 | 내용 |
|---|---|
| 표준 번호 | IEC 60601-1-2:2014+AMD1:2020 (Edition 4.1) |
| 발행일 | 2020-09-01 (A1 통합 버전) |
| EU 동등 | EN 60601-1-2:2015+A1:2021 |
| 적용 범위 | 의료전기기기(ME equipment) 및 ME 시스템의 EMC |
| 상위 표준 | IEC 60601-1:2005+A1+A2 (Ed. 3.2) |
| 핵심 접근 | **Risk management 기반** — 고정 시험 레벨이 아닌 Risk Management File과 연계하여 Pass/Fail 기준 설정 |

**Edition 4.1 vs 4.0 주요 차이 (A1:2020 도입 15개 변경사항 핵심)**

- 신규 시험: **Proximity magnetic field immunity** (Subclause 8.11, IEC 61000-4-39)
- 확장: Conducted immunity → SIP/SOP 케이블 1m 초과 시 적용
- 변경: Conducted emissions (CISPR 11) → 최소·최대 정격전압 양쪽에서 수행
- 참조 표준 갱신: IEC 61000-4-5:2014+A1:2017, IEC 61000-4-11+A1:2017, CISPR 11:2015+A1+A2, ISO 14971:2019

## 3. 환경 분류 (EMC 시험 레벨 결정 기준)

IEC 60601-1-2 Ed. 4.1은 **의도된 사용 환경(Intended Use Environment)**에 따라 시험 레벨을 구분한다.

| 환경 코드 | 명칭 | 설명 | 자사 제품 해당 여부 |
|---|---|---|---|
| **PHE** | Professional Healthcare Facility Environment (전문 의료기관 환경) | 병원·클리닉·진단실 등 | X-ray Detector ✅, Handheld Source ✅, GUI SW ✅ |
| **HHE** | Home Healthcare Environment (가정 의료 환경) | 자택 사용 | 해당 없음 (자사 3개 제품 모두 PHE 전용) |
| **Special** | Special Environments | 구급차·항공기 등 | 해당 없음 |

> **자사 3개 제품은 모두 PHE(전문 의료기관) 환경으로 분류된다.** 하단 표의 PHE 열 레벨을 기준으로 적합성을 입증해야 한다.

## 4. Emission (방사·전도 방해) 시험 항목

**참조 표준: CISPR 11 (ISM 기기 방사 특성)**

### 4-1. Group·Class 분류

| 분류 | 정의 |
|---|---|
| **Group 1** | 의도적 ISM 주파수 RF 발생 없음 → 자사 X-ray Detector, Handheld X-ray Source, GUI SW 모두 해당 |
| Group 2 | 의도적 ISM 주파수 RF 사용 (예: RF 치료기) |
| **Class A** | PHE 전용 또는 대형·고정 장비 → 자사 3개 제품 기본 분류 |
| Class B | 가정용 환경 사용 가능 |

### 4-2. Emission 시험 항목 및 Limit

| 시험 항목 | 참조 표준 | 주파수 범위 | Limit (Group 1, Class A) | 비고 |
|---|---|---|---|---|
| **Radiated Emission** | CISPR 11 | 30 MHz ~ 1 GHz (측정거리 10 m) | 30–230 MHz: 40 dBμV/m; 230 MHz–1 GHz: 47 dBμV/m | 30 m 거리 등가 환산 가능 |
| **Conducted Emission** (AC 전원선) | CISPR 11 | 0.15 MHz ~ 30 MHz | QP: 79 dBμV (0.15–0.5 MHz), 73 dBμV (0.5–30 MHz); AVG: 66/60 dBμV | **A1:2020 변경:** 최소 정격전압 + 최대 정격전압 각각 수행 |
| **Conducted Emission** (DC 전원선, 해당 시) | CISPR 11 | 0.15 MHz ~ 30 MHz | 동상 Class A Limit 동일 적용 | |

> **A1:2020 핵심 변경**: 이전 Ed. 4.0은 공칭 전압 1개만 시험했으나, Ed. 4.1부터 최소·최대 정격전압 양쪽에서 Conducted Emission 수행 필수.

### 4-3. 대형 고정 장비 (Large ME Equipment) 예외

ME 장비가 2m × 2m × 2.5m 부피를 초과하는 경우 → 현장 설치 후 In-situ 시험 또는 제조사 premises에서 시험 허용. 자사 X-ray Detector 및 GUI SW 워크스테이션은 해당 없음(소형). **Handheld Source**는 휴대형으로 해당 없음.

## 5. Immunity (내성·면역) 시험 항목 및 Pass Criteria

### 5-1. 시험 레벨 개요 (PHE 기준)

| 시험 항목 | 참조 표준 | Port | PHE 시험 레벨 | 비고 |
|---|---|---|---|---|
| **ESD (정전기 방전)** | IEC 61000-4-2 | 인클로저 | Contact ±6 kV, Air ±8 kV | 인체 접촉 가능 모든 표면 |
| **EFT/Burst (전기고속과도)** | IEC 61000-4-4 | AC 전원 | ±2 kV | |
| | | DC 전원 | ±2 kV | |
| | | SIO 케이블 >3m | ±1 kV | |
| **Surge (서지)** | IEC 61000-4-5 | AC 전원 (L-E) | ±2 kV | IEC 61000-4-5:2014+A1:2017 적용 |
| | | AC 전원 (L-L) | ±1 kV | |
| | | DC 전원 (L-E) | ±0.5 kV | |
| **Conducted RF Disturbance** | IEC 61000-4-6 | AC/DC 전원 | 3 Vrms (150 kHz–80 MHz) | 케이블 길이 >3m 적용 |
| | | SIP/SOP >1m **(A1:2020 신규)** | 3 Vrms | **Ed. 4.0에서는 미적용** |
| **Radiated RF** | IEC 61000-4-3 | 인클로저 | 3 V/m (80 MHz–2.5 GHz) | 900 MHz±5 MHz: 27 V/m (특수 레벨) |
| **Power Freq. Magnetic Field** | IEC 61000-4-8 | 인클로저 | 30 A/m (PHE) | 50 Hz 또는 60 Hz (장비 사용 전원 주파수와 동일) |
| **Voltage Dips** | IEC 61000-4-11 | AC 전원 | 0%/0.5 cycle; 0%/1 cycle; 40%/6 cycle; 70%/25 cycle | 최소·최대 정격전압 양쪽 수행 (A1:2020) |
| **Voltage Interruptions** | IEC 61000-4-11 | AC 전원 | 0%/250 cycle | |
| **Proximity Magnetic Field** **(A1:2020 신규)** | IEC 61000-4-39 | 인클로저 (자기감응 부품 해당 시) | 30 kHz: 8 A/m (CW); 134.2 kHz: 65 A/m (PM 2.1 kHz); 13.56 MHz: 7.5 A/m (PM 50 kHz) | 아래 §5-2 상세 참조 |

> **900 MHz 강화 레벨**: 900 MHz±5% 대역 27 V/m (DECT 전화기 등 무선기기 고려). 일부 시험소는 이 레벨에서 별도 시험 수행.

### 5-2. Proximity Magnetic Field Immunity (IEC 61000-4-39) 상세 — A1:2020 신규

| 주파수 | 변조 방식 | 시험 레벨 | 해당 소스 |
|---|---|---|---|
| 30 kHz | CW (무변조) | 8 A/m | 무선 충전기, NFC 전원 |
| 134.2 kHz | Pulse 변조 (2.1 kHz, 50% duty) | 65 A/m | 동물/사람 이식형 RFID 리더기 |
| 13.56 MHz | Pulse 변조 (50 kHz, 50% duty) | 7.5 A/m | NFC, RFID 리더기 |

**면제 조건 (Exemption, Subclause 8.11)**
1. ME 장비 인클로저 내부 또는 부속 액세서리에 자기감응 소자가 없는 경우
2. 자기감응 소자가 소스와 충분한 이격 거리 확보 입증 (RISK 관점)
3. RISK 수용 가능 입증 (Risk Management 문서 포함)

> 면제 시에도 의사결정 근거는 기술문서에 **반드시 기록** 필요.

**자사 제품 해당성 검토**:
- X-ray Detector: Hall effect 센서, 신호 변환 회로 포함 가능 → **시험 필요 가능성 높음** [검증 필요]
- Handheld X-ray Source: 내부 고전압 회로, 제어 MCU → 자기감응 소자 여부 확인 필요 [검증 필요]
- GUI SW 워크스테이션: 일반 PC 기반 시 자기감응 소자 최소화 → 면제 가능성 있으나 확인 필요 [검증 필요]

## 6. Pass/Fail 기준 (Acceptance Criteria)

IEC 60601-1-2 Ed. 4.1은 기존 IEC 61000-4-x 시리즈의 Criterion A/B/C를 그대로 채택하지 않는다. 대신 **제조사가 Risk Management 프로세스(ISO 14971:2019)와 연계하여 기준을 정의**한다.

### 6-1. 기본 개념

| 개념 | 정의 |
|---|---|
| **Basic Safety** | 불합리한 물리적 위험 부재 (예: 감전, 화재) |
| **Essential Performance** | 제조사가 정의한, 기기 안전·의도된 사용에 필수적인 기능 |
| Pass 조건 | EMC 시험 중·후 **Basic Safety 유지 + Essential Performance 유지** |
| Fail 조건 | Basic Safety 손상 OR Essential Performance 손실 |

### 6-2. Essential Performance 정의 예시 (자사 3개 제품)

| 제품 | Essential Performance 후보 |
|---|---|
| X-ray Detector | 영상 취득 기능 유지, 노출 신호 정확 처리, 환자 선량 지시 정확도 |
| Handheld X-ray Source | kV·mA 설정 정확도, 노출 인터록 동작, 방사 방호 지시 기능 |
| GUI SW | 촬영 지시·취득 명령 정확 전달, 환자/영상 데이터 무결성 |

### 6-3. 합부 판정 운영 원칙

1. 시험 계획서에 Essential Performance 판정 기준과 모니터링 방법 사전 문서화
2. 시험 중 성능 저하 발생 시 → 스스로 복구(auto-recovery) 가능하면 합격 가능
3. 오퍼레이터 개입 없이 영구적 기능 손실 → 불합격
4. Risk Management File과 시험 보고서 상호 참조 필수

## 7. Emission 및 Immunity 시험 절차 요약 (실무 흐름)

```
[1] 사용 환경 결정 (PHE / HHE / Special)
    ↓
[2] Emission: CISPR 11 Group·Class 결정
    → Radiated + Conducted (최소·최대 정격전압 × 2회)
    ↓
[3] Immunity 계획 수립
    → 포트별 시험 레벨 확정 (PHE 표준 레벨 또는 Risk 기반 조정)
    → Essential Performance 판정 기준 문서화
    ↓
[4] Proximity Magnetic Field 면제 여부 판단 (Subclause 8.11 flowchart)
    → 자기감응 소자 목록 확인 → 면제 or 시험 결정 기록
    ↓
[5] 시험 수행 (공인 시험기관 또는 내부 EMC 설비)
    ↓
[6] 시험 보고서 작성 (IEC 60601-1-2 요구사항 준수)
    → 시험 레벨, 구성, 판정 기준, 결과, Deviation 기록
    ↓
[7] 기술문서(Technical File / DHF) 포함
```
