# IEC 60601-2-54:2022 촬영·투시 X선 장비 시험항목 매트릭스

## 1. 표준 개요

| 항목 | 내용 |
|---|---|
| 표준 번호 | IEC 60601-2-54:2022 (Edition 2.0, 2022-09-26) |
| 정식 명칭 | Medical electrical equipment – Part 2-54: Particular requirements for the basic safety and essential performance of X-ray equipment for radiography and radioscopy |
| 적용 범위 | 투영 촬영(projection radiography) 및 간접 투시(indirect radioscopy) 용 의료용 X선 장비 및 시스템 |
| 대체 판 | IEC 60601-2-54:2009 (Ed.1) + AMD1:2015 + AMD2:2018 폐지·대체 |
| 상위 표준 | IEC 60601-1:2005+AMD1:2012+AMD2:2020 (General Standard) |
| Collateral | IEC 60601-1-2 (EMC), IEC 60601-1-3 (방사선방호), IEC 60601-1-11/-12 (제조사 선언 시 적용) |
| 연관 표준 | IEC 60601-2-28:2017 (X선관 조립체), IEC 60601-2-43:2022 (중재시술), IEC 61223-3-8:2024 (QC 시험) |
| TC/SC | IEC TC 62 / SC 62B (Diagnostic imaging) |
| 표준 유형 | Particular Standard (PS) |
| IECEE TRF | 60601-2-54I:2023 (CB Scheme 시험보고서 양식) |

### 1.1 적용 대상 / 적용 제외

| 구분 | 내용 |
|---|---|
| **적용 대상** | 일반 X선 촬영 장치, 디지털 방사선 촬영(DR), 투시 장치(fluoroscopy), 이동형 X선 장치, 수술실 C-arm (비중재시술 용도) |
| **적용 제외** | CT (IEC 60601-2-44), 유방촬영 (IEC 60601-2-45), 치과용 (IEC 60601-2-65), 중재시술 C-arm (IEC 60601-2-43), 골밀도 측정, 방사선치료, 방사선치료 시뮬레이터 |

> **주의:** Interventional 목적 C-arm은 IEC 60601-2-43이 적용되며, IEC 60601-2-54 요건도 교차 참조됨.

### 1.2 Edition 2 주요 변경 사항 (vs Ed.1.2:2018)

| 변경 구분 | 상세 |
|---|---|
| 용어 통일 | DOSEMETER → **DOSIMETER** (용어 정규화) |
| 조항 재배치 | IEC TR 60788:2004 인용 용어·정의를 Clause 201.3으로 이동 |
| Collateral 확장 | IEC 60601-1-11 (홈케어) 및 IEC 60601-1-12 (구급 환경) — 제조사 선언 시 적용 |
| 조항 삭제 | **201.11.101** (X선관 과온도 방호) 삭제 → IEC 60601-1 + IEC 60601-2-28:2017로 완전 이전 |
| 표시 명확화 | 인터록 표시등(indicator light) 요건 구체화 |
| 용량 구조 | 에너지 등가 공기커마(Energy equivalent air kerma) 측정 절차 갱신 |

## 2. 3개 지역 규제 적용 현황

| 구분 | MFDS (한국) | FDA (미국) | MDR (EU) |
|---|---|---|---|
| 채택 표준 | KS C IEC 60601-2-54 (동등 채택) | IEC 60601-2-54 FDA Recognized Consensus Standard | EN IEC 60601-2-54:2022 (BSI, CLC/TC 62 발행) |
| 추가 법령 | 의료기기 기준규격 (MFDS 고시) | **21 CFR Part 1020** (EPRC): §1020.30 (일반) + §1020.32 (투시) + §1020.33 (촬영) | MDR Annex I GSPR Chapter II (방사선 방호) + Council Directive 2013/59/Euratom (BSS) |
| 최대 선량률 (투시) | ≤ 100 mGy/min (air kerma, IEC 기준) | **≤ 87.6 mGy/min** (≈10 R/min, 정상) / **≤ 175.2 mGy/min** (High-Level 모드) (§1020.32) | ≤ 100 mGy/min (IEC 기준, EN 동일) |
| 투시 타이머 | 5분 경보 (IEC 요건) | **5분 경보 필수** (§1020.32(d)) | 5분 경보 (IEC 기준) |
| 제출 서류 | 기술문서 + 국내 공인시험기관 성적서 | eSTAR Performance Testing + DoC (IEC 60601-2-54 인용) | Annex II TD + NB 심사 |
| Harmonized 여부 | KS 동등 채택 | FDA Consensus Standard 인정 | EU OJ 게재 EN IEC 60601-2-54:2022 |

> **핵심 차이:** FDA §1020.32는 투시 최대 선량률을 **87.6 mGy/min** (IEC 100 mGy/min보다 약 13% 엄격)으로 규정.  
> 글로벌 출시 전략: **FDA 기준(≤ 87.6 mGy/min)으로 설계** 후 IEC/MDR/MFDS 자동 충족 가능.

### 3.2 Clause 203.5 — 투시 (Radioscopy) 선량 요건 ★ 최중요

#### 3.2.1 최대 Air Kerma Rate (203.5.101)

| 구분 | 정상 투시 | High-Level 투시 |
|---|---|---|
| **IEC (MFDS/MDR)** | **≤ 100 mGy/min** | **≤ 200 mGy/min** |
| **FDA (21 CFR 1020.32)** | **≤ 87.6 mGy/min** (≈10 R/min) | **≤ 175.2 mGy/min** (≈20 R/min) |
| 측정 조건 | 환자 입사면 기준 (Isocenter 또는 표면 15 cm) | 특별 활성화 후 측정 |
| 활성화 방식 | 일반 풋 스위치 | 별도 조작(지속 압박 + 청각 경보) |
| 조작자 경보 | 5분 경과 시 청각 신호 (§203.5.103) | High-Level 작동 중 연속 청각 경보 |

> **자사 Handheld X-ray Source 적용:** 투시 기능 미포함 시 203.5 조항 해당 없음. 촬영 전용 확인 필요.  
> **자사 촬영실 GUI SW 적용:** 투시 타이머 표시(203.5.103) 및 High-Level 경보 UI 구현 필요.

### 3.3 Clause 203.6 — AEC (Automatic Exposure Control) ★

| 시험 항목 | 측정 방법 | Pass Criteria |
|---|---|---|
| AEC 목표 달성 정확도 | 표준 팬텀(water equivalent) 두께 변화 시 Air Kerma 측정 | 목표값 대비 **±25%** 이내 |

| 소아 mode (해당 시) | 소아 팬텀(5 cm) 적용 AEC 동작 | 정상 AEC 동작 확인 |

### 3.4 Clause 203.7 — Exposure Index (디지털 방사선 촬영) ★

IEC 62494-1:2022 기반 DR 노출지수. X-ray Detector 및 GUI SW 중요 적용 항목.

| 파라미터 | 정의 | 요건 |
|---|---|---|
| **EI (Exposure Index)** | 영상수용체 실제 노출량 지수 | 표시 정확도 ±15% |
| **EIT (Target EI)** | 목표 노출지수 | 시스템별 설정값, IFU에 명시 |
| **DI (Deviation Index)** | DI = 10 × log₁₀(EI/EIT) | DI ≤ ±1 (정상 노출 범위) 표시 |
| **표시 요건** | EI/DI를 영상과 함께 표시 | 표시 위치·방법 IFU 명시 |

> **자사 X-ray Detector:** 영상수용체 캘리브레이션에 따른 EI 계산 정확도가 핵심.  
> **자사 GUI SW:** EI/DI 표시 UI, 목표 EI 설정 기능 구현 필수.

### 3.5 Clause 203.8 — 환자 선량 표시 ★

| 파라미터 | 약어 | 표시 정확도 요건 | 적용 |
|---|---|---|---|
| Dose Area Product | DAP (Gy·cm²) | **±35%** (측정값 대비) | 투시 + 촬영 |
| Cumulative Air Kerma | AK (mGy) | **±35%** (측정값 대비) | 투시 장비 |
| Air Kerma Rate | AKR (mGy/min) | **±35%** (실시간 표시) | 투시 |
| Kerma Area Product | KAP | DAP와 동일 개념, 표준 내 혼용 | — |

> FDA §1020.32(k): 2006년 이후 제조 투시 장비는 누적 AK 표시 필수.  
> **자사 GUI SW:** DAP 미터 인터페이스 및 누적 AK 표시 화면 구현 필수.

### 3.6 Clause 201.12 — 제어 정확도

| 파라미터 | 허용 오차 | 측정 조건 |
|---|---|---|
| Peak tube voltage (kVp) | **±10%** | 설정 kVp vs 실측 (반도체 kVp meter) |
| Tube current (mA) | **±20%** (연속); **±20%** (투시) | mA·s 측정 |
| Exposure time | **±10%** (>0.1 s); **±20%** (≤0.1 s) | 타이머 측정 |
| mAs (곱) | **±20%** | mA × time |
| Air Kerma 출력 재현성 | CV **≤ 5%** | 동일 조건 5회 |

### 3.7 Clause 203.9 — 빔 제한 장치 (Collimator)

### 6.2 CB Scheme 활용

IEC 60601-2-54는 IECEE CB Scheme 적용 가능. TRF 60601-2-54I:2023 사용.  
- CB Certificate 취득 시 MFDS·MDR·FDA 각국 인증 기관 제출 가능.  
- 국내 공인시험기관(KTL, KTR 등)에서 CB Report 발행 가능.  
- **FDA:** IECEE CB Report + 510(k) 제출 시 일부 시험 면제 불가 (FDA는 독자 검토); DoC 대신 시험 데이터 직접 제출 권장.
