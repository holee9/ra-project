# IEC 60601-1-3:2008+AMD1:2013+AMD2:2021 방사선 방호 시험항목 매트릭스

## 1. 표준 개요 및 이력

### 1.1 적용 범위 (Scope)

진단용 X선 기기 및 그 서브어셈블리에 적용.  
목표: ALARA(As Low As Reasonably Achievable) 원칙에 따라 환자·조작자·직원·일반인의 피폭을 최소화하되, 방사선학적 시술의 이득을 저해하지 않는 일반 요건 수립.

**적용 대상 기기 예:**
- 일반 촬영장치 (Radiographic equipment)
- 투시장치 (Fluoroscopic equipment)
- Handheld / 이동형 X선 발생기
- X선 CT (CT는 IEC 60601-2-44와 병행 적용)
- 유방촬영장치 (IEC 60601-2-45와 병행 적용)
- 치과용 X선 (IEC 60601-2-65 등과 병행 적용)

**적용 제외:** 치료용 방사선 장치 (IEC 60601-2-1 등 별도 적용)

### 2.2 Clause 12.2.101 — DSA 누설방사선 (Leakage Radiation)

**시험 목적:** X선관 하우징에서 유용빔 이외 방향으로 새는 방사선 제한

| 항목 | 요건 |
|---|---|
| **한도값** | **≤ 1 mGy** 공기커마 / 1시간 (1 m 거리 전방향) |
| 시험 조건 | 최대 정격 연속 관전류 (Leakage Technique Factors) |
| 측정 위치 | 초점(focus)에서 1 m, 전방향 |
| 측정 평균 면적 | 100 cm² (최대 선 치수 ≤ 20 cm) |
| 빔 차폐 | 콜리메이터 완전 폐쇄 + 납 차폐판 사용 |
| 비교 기준 (FDA) | 21 CFR 1020.30(k): **≤ 0.88 mGy/h** (IEC보다 엄격) |

> **주의 — 지역별 차이:** IEC 60601-1-3은 1 mGy/h, 21 CFR 1020.30은 0.88 mGy/h.  
> 미국 판매 제품은 0.88 mGy/h 기준으로 설계·시험해야 함.

### 2.3 Clause 12.2.102 — DSA 이외 구성요소 방사선

| 항목 | 요건 |
|---|---|
| **한도값** | **≤ 20 μGy/h** 공기커마, 접근 가능 표면 5 cm |
| 적용 대상 | 콜리메이터, 필터 어셈블리, 이미지 리셉터 등 |
| 비교 기준 (FDA) | 21 CFR 1020.30(l): **≤ 18 μGy/h** (유사, 단위 환산 차이) |
| 측정 평균 면적 | 100 cm² (최대 선 치수 ≤ 20 cm) |

### 2.4 Clause 12.4.101 — Beam Limiting Device (콜리메이터)

| 시험 항목 | 요건 |
|---|---|
| 조사야 위치 정확도 | 방사선조사야 경계 ≤ 이미지 리셉터 크기의 ±2% (SID 기준) |
| 빛조사야 일치도 | 빛조사야↔방사선조사야 불일치 ≤ ±2% SID |
| 최소 조사야 | 기기 사양에 따른 최소 콜리메이션 가능 여부 |
| 원격/자동 콜리메이션 | 해당 시 소프트웨어 구동 정밀도 확인 |

### 2.5 Clause 13.2.101 — HVL (Half-Value Layer)

빔 품질을 정의하는 핵심 시험. 환자 피폭 저감에 직결.

| kVp (측정값) | 최소 HVL (mm Al) — IEC 60601-1-3 기준 |
|---|---|
| 30 | 0.3 |
| 40 | 0.4 |
| 50 | 0.5 |
| 60 | 1.5 |
| 70 | 1.8 |
| 80 | 2.9 |
| 90 | 3.2 |
| 100 | 3.6 |
| 110 | 3.9 |
| 120 | 4.3 |
| 130 | 4.7 |
| 140 | 5.0 |
| 150 | 5.4 |

> HVL 시험 방법: 이온전리조(ionization chamber) + Al 여과판 순차적 삽입 → 공기커마 50% 감소 두께 측정.  
> 참조: IEC 60601-1-3 §13.2.101 및 IPEM Report 32(시험 절차).

## 4. 자사 3개 제품 영향 맵핑

### 4.1 X-ray Detector (X선 검출기)

| Clause | 적용 여부 | 비고 |
|---|---|---|
| 12.2.101 (DSA 누설방사선) | **간접 적용** | Detector는 방사선 발생원이 아님. 단, Detector를 포함한 **시스템** 전체에서 검토 필요. 단독 제품 인허가 시 해당 없음 |
| 12.2.102 (DSA 이외 구성요소) | **해당 없음** | Detector는 X선을 발생시키지 않음 |
| 12.4.101 (Beam Limiting) | **해당 없음** | 단독 Detector에는 콜리메이터 없음 |
| 13.2.101 (HVL) | **해당 없음** | Source 측 요건 |
| 16.101 (Al 등가치) | ⚠️ **적용 가능** | Detector 전면 커버(패널) 소재의 Al 등가치 — 환자와 Detector 사이 구조물 흡수 최소화 요건 |
| §10 (Information for Safety) | **적용** | Detector 사용 시 피폭 저감 지침, IFU 기재 요건 |

**행동 지시:**
- Detector 전면 패널 소재의 **Al 등가치** 측정 및 IEC 16.101 적합성 확인 수행
- 시스템 통합 시험 시 Source 제조사와 누설방사선 시험 책임 분담 명확화 (DHF에 기재)

---

### 4.2 Handheld X-ray Source (소형 핸드헬드 X선 발생기)

**IEC 60601-1-3의 가장 직접적인 적용 대상 제품.**

| Clause | 적용 여부 | 행동 지시 |
|---|---|---|
| **12.2.101 (DSA 누설방사선)** | ✅ **필수** | **MFDS·MDR: ≤ 1 mGy/h / FDA: ≤ 0.88 mGy/h** @ 1 m — 설계 단계부터 Tube Housing 차폐 설계에 반영 |
| **12.2.102 (구성요소 방사선)** | ✅ **필수** | 핸드헬드 그립, 전자회로 외장 등 DSA 외 부품 방사선 ≤ 20 μGy/h (FDA: 18 μGy/h) |
| **12.4.101 (Beam Limiting)** | ✅ **필수** | 내장 콜리메이터 정밀도 시험 — 조사야 정확도 ≤ ±2% SID |
| **13.2.101 (HVL)** | ✅ **필수** | 사용 kVp 범위 전체 HVL 측정 (예: 60~70 kVp 위주) — 최소 Al 필터 설계 |
| **13.3.101 (Focal Spot)** | ✅ **필수** | 공칭 초점 크기 IEC 60336 기준 시험 |
| **§7.2.13 (IFU 정보)** | ✅ **필수** | Leakage 기술인자, 최소 사용거리, 산란방사선 주의 사항 IFU 기재 |
| **§7.4.3 (표시)** | ✅ **필수** | 방사선 발생 경고 표시, 조사야 지시기 |

**추가 고려 (Handheld 특이사항):**
- AMD2:2021에서 **portable/handheld 기기**에 대한 운용 요건 명확화 → AMD2 전문 검토 필수 [검증 필요]
- 배터리 구동 → **충전 중 X선 발생 방지** 회로 요건 확인 (IEC 60601-1 §11 연계)
- 핸드헬드 특성상 조작자가 1 m 이내 위치 가능 → 누설방사선 시험 시 근거리 측정도 DHF에 추가 권고

### 4.3 촬영실 GUI SW

| Clause | 적용 여부 | 비고 |
|---|---|---|
| 12.2.101 / 12.2.102 | **해당 없음** | SW 자체는 방사선 발생 없음 |
| 12.4.101 (Beam Limiting) | ⚠️ **간접 적용** | GUI에서 콜리메이터 원격 제어 기능 구현 시 — SW 명령 정밀도·안전 인터락 요건 |
| **§7.2.13 (IFU 정보)** | ✅ **적용** | GUI SW가 선량 정보(DAP, AKR)를 표시하는 경우 — 표시 정확도·형식 요건 |
| **선량 표시 기능** | ✅ **적용** | 투시장치 연동 GUI의 경우 공기커마율(AKR) 표시 요건 (IEC 60601-2-43/2-54 연계) |
| AMD2:2021 선량 모니터링 | ⚠️ **검토 필요** | 환자선량 모니터링 기능 SW 구현 시 표시 정확도 요건 적용 가능 |
| IEC 62304 SW 수명주기 | ✅ **적용** | 콜리메이터 제어·선량 계산 알고리즘은 Safety Class B 이상으로 관리 |

**행동 지시:**
- GUI SW가 선량(DAP, AKR, mAs) 표시 기능 포함 시 IEC 60601-1-3 §7.2.13 및 관련 Particular Standard 선량 표시 요건 검토
- 콜리메이터 원격 제어 SW 구현 시 안전 인터락 (Fail-safe) 요건을 IEC 62304 Safety Class B 이상으로 관리
