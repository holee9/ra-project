# ISO 13485:2016 절차서 목록 (섹션 4~8) 및 뼈대

## 1. 개요

ISO 13485:2016은 의료기기 QMS 국제표준으로, 3개 지역에서 다음과 같이 연계된다.

| 지역 | 적용 방식 | 최신 현황 |
|---|---|---|
| **MFDS (KGMP)** | ISO 13485:2016 기반, 별도 KGMP 인증 필수 | MFDS 고시 제2025-22호 (2025-04-07) 개정 — SaMD 조항 삭제, Class 3 심사 강화 |
| **FDA (QMSR)** | 21 CFR Part 820 개정으로 ISO 13485:2016 인용 통합 | 2026-02-02 발효. ISO 13485 준수만으로 완전 충족 불가 — FDA 추가요건 별도 이행 필요 |
| **MDR** | EN ISO 13485:2016 조화표준으로 NB 심사 기준 | Annex IX QMS 심사 직접 연계 |

---

## 2. 절차서 분류 원칙

| 구분 | 설명 |
|---|---|
| **필수(M)** | 표준 조항에서 "documented procedure shall be established" 명시 |
| **권고(R)** | 조항에서 records 또는 documented information 요구 → 절차 없으면 심사 지적 위험 |
| **적용 조건부(C)** | 조직 활동에 해당 프로세스 존재 시에만 필요 |

## 6. 3개 지역 비교 매트릭스

| 요건 항목 | MFDS (KGMP) | FDA (QMSR) | MDR |
|---|---|---|---|
| 기준 표준 | ISO 13485:2016 기반 KGMP 고시 (2025-22) | ISO 13485:2016 인용 통합 (21 CFR 820, 2026-02-02) | EN ISO 13485:2016 조화표준 |
| 인증 방식 | MFDS 지정기관 심사 (KGMP 인증서 별도 필요) | FDA QSI 심사 / MDSAP 인정 | NB의 Annex IX QMS 심사 |
| 설계관리 적용 | 제조 SOP + 설계이관 기록 핵심 | QMSR §820.30 — Design Controls 유지 (ISO 7.3 동일) | NB 심사 시 D&D 파일 전체 제출 |
| SaMD 적용 | 디지털의료제품법으로 분리 (2025-01-24) | QMSR + FDA SW Guidance (2024-06) 병행 | MDR + MDCG 2019-11 병행 |
| CAPA 요건 | CA/PA 절차서 + 효과검증 기록 | 동일 (ISO 8.5.2/8.5.3) | 동일 + PMCF 연계 |
| 불만 처리 타임라인 | 이상사례 15일(사망/중상) / 30일(기타) | MDR: 30일(사망/중상), 5일(즉각) | Vigilance: 15일(사망·중상) / 30일(기타) |
| 내부심사 주기 | 최소 1회/년 | 동일 | 동일 |
| 공급업체 관리 | 구매 승인업체 목록 (ASL) 필수 | 동일 | UDI-DI 연계 EUDAMED 등록 고려 |

## 7. 자사 3개 제품 영향 맵핑

### 7.1 X-ray Detector
- **핵심 절차서**: SOP-CAL-001 (교정 — DQE/MTF 측정 장비), SOP-PRD-001 (생산 관리), SOP-PRD-004 (제품 식별), SOP-COM-001 (불만 처리)
- **주의점**: 검출기 성능은 생산 환경(온도·습도·전자기 차폐)에 민감 → WI-ENV-001과 연계 필수

### 7.2 Handheld X-ray Source
- **핵심 절차서**: SOP-PRD-002 (공정 밸리데이션 — 방사선 출력 재현성), SOP-PRD-005 (추적성 — 배터리·방사선 발생관 로트 추적), SOP-COM-002 (규제기관 보고 — 방사선 이상사례)
- **주의점**: 21 CFR 1020.30 방사선 성능 인증 관련 생산 기록은 SOP-PRD-001과 연계

### 7.3 촬영실 GUI SW (SaMD)
- **핵심 절차서**: SOP-DND-001 (설계 관리 — IEC 62304 SDP와 통합), SOP-CAPA-001/002 (Bug/Change → CAPA 연계), SOP-COM-001 (SW 결함 불만 처리)
- **주의점**: MFDS: 디지털의료제품법 별도 적용 → SOP-DND-001에 디지털의료기기 특이사항 섹션 추가 필요. FDA: QMSR §820.30 + SW 가이던스(2024-06) 동시 준수
