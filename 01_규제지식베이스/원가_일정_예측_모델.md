## 6. 일정 로드맵 (3지역 동시 진행 예시)

X-ray Detector 기준 (가장 복잡한 시험 요건).

```
Month  1– 3: 사전 준비 — 등급 확정, NB 선정, FDA Pre-Sub 제출, MFDS 사전상담
Month  3– 8: 시험 진행 — IECEE CB 인정 랩에서 60601-1, EMC, 60601-1-3, DQE 동시 진행
Month  6– 9: 기술문서 작성 — Common Core TD (STED 기반), 지역별 번역·보완
Month  9–10: MFDS 제출 → 심사 (55–70 영업일)
Month 10–12: FDA 510(k) 제출 → 심사 (90–175 FDA Days)
Month  9–18: MDR NB 심사 → Class IIa CE 마킹
Month 12–14: MFDS 허가 취득 (예상)
Month 14–18: FDA Clearance 취득 (예상)
Month 18–22: CE 마킹 취득 (예상)
```

> Handheld X-ray Source (MDR Class IIb)는 전체 기간 +4–8개월 추가.  
> GUI SW는 전체 기간 3–6개월 단축 가능.
