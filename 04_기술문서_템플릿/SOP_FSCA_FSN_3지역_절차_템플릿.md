# FSCA / FSN 절차·템플릿 (3개 지역)

## 1. 개요

| 용어 | 정의 |
|---|---|
| **FSCA** (Field Safety Corrective Action) | 이미 시장에 출시된 기기와 관련하여 사망 또는 건강 상태의 심각한 악화 위험을 줄이기 위해 취하는 안전 예방·교정 조치 |
| **FSN** (Field Safety Notice) | FSCA를 수행할 때 기기 사용자/고객에게 발송하는 공식 통보문 |
| **Recall** (FDA) | 법령 위반 제품의 시장 회수·교정 조치 (21 CFR 7) |
| **Correction** (FDA) | 기기를 다른 위치로 물리적 이동 없이 현장에서 수행하는 수리·수정·조정 |
| **Removal** (FDA) | 기기를 현재 위치에서 다른 장소로 물리적으로 이동시키는 조치 |
| **회수** (MFDS) | 의료기기법 제31조 기반, 결함 기기의 시장 회수 |

## 2. 3개 지역 비교 매트릭스

| 항목 | MDR (EU) | FDA (미국) | MFDS (한국) |
|---|---|---|---|
| **근거 법령** | MDR (EU) 2017/745 Art.83, 87, 89 | 21 CFR 806 (Correction/Removal), 21 CFR 7 (Recall) | 의료기기법 제31조·제32조, 시행규칙 제51조 |
| **보고 대상** | 제조업체 | 제조업체·수입업체 | 제조업체·수입업체 |
| **보고 시점** | FSCA 착수 **전** 사전 보고 (긴급 시 착수와 동시) | 착수 후 **10영업일 이내** | 인지 후 **즉시** (지체 없이) |
| **보고 수신처** | EUDAMED + 해당 관할 기관 | FDA ORA 구역별 Recall Coordinator (DRC) | 식품의약품안전처장 |
| **보고 형식** | MIR Form 7.3.1 (2024-12 이후 최신 버전) + FSCA Form | Form FDA 5072 (Correction/Removal Report) | 의료기기 이상사례 보고서 (식약처 소정 양식) |
| **FSN 사전 검토** | 관할 기관에 초안 제출 후 의견 수렴 (긴급 제외) | Recall letter 형태로 DRC와 사전 협의 권장 | 회수계획서 제출 후 승인 |
| **등급 분류** | N/A (심각도별 FSCA 유형 구분) | Class I / II / III | 1~3단계 (의료기기법 시행규칙) |
| **기록 보존** | Technical Documentation에 포함 (PMS 파일) | 기기 예상 수명 + 2년 | 관련 법령 기간 준수 |
| **공개 여부** | EUDAMED 공개 모듈 | FDA Enforcement Report (주간 공개) | 식약처 홈페이지 공개 |

## 3. FSCA 발동 판단 기준

### 3.1 FSCA 발동 필요 상황
1. **심각한 이상사례(Serious Incident)** 원인이 기기 결함으로 확인된 경우
2. 사망·중대한 건강 악화 위험을 초래할 **잠재적 결함** 발견 시
3. **Vigilance 조사** 결과 예방적 FSCA 필요 판정 시
4. 규제 기관으로부터 **FSCA 착수 요청** 수신 시

### 3.2 FSCA 유형 분류 (MDR 기준)
| 유형 | 설명 |
|---|---|
| Recall (회수) | 기기를 사용자로부터 물리적으로 회수 |
| Device modification | 현장 소프트웨어 업데이트, 하드웨어 수리·조정 |
| Advice of use | 사용 제한·금지·조건 변경 안내 |
| Labeling change | 표시기재·IFU 개정 및 재발행 |
| Customer notification only | 안전 정보 통보 (조치 불요) |

### 3.3 자사 3개 제품 위험 시나리오

| 제품 | 주요 FSCA 트리거 시나리오 |
|---|---|
| **X-ray Detector** | DQE 성능 허용 범위 이탈 (이미징 오류 → 오진 위험), 기계적 파손으로 전기 안전 침해, 환자 접촉부 생물학적 오염 |
| **Handheld X-ray Source** | 방사선 출력 허용치 초과 (선량 과다), 배터리 열 폭주 위험, 충전 회로 결함으로 전기 안전 침해 |
| **촬영실 GUI SW** | 이미지 처리 오류 (Artifact → 진단 오류), 보안 취약점으로 무단 접근·조작, 선량 계산 오류 |

## 6. FSCA 보고 양식 요약 (규제 기관 제출용)

### 6.1 MDR — EUDAMED FSCA Report 필수 항목
| 항목 | 내용 |
|---|---|
| Device identification | UDI-DI, 모델명, LOT/Serial |
| Description of FSCA | 조치 유형 및 이유 |
| Risk assessment | 위험 발생 확률, 심각도, 추정 영향 기기 수 |
| FSN reference | FSN 번호 및 발행일 |
| Timeline | 착수일, 완료 예상일 |
| Distribution countries | 영향받는 EU 회원국 목록 |

### 6.2 FDA — Form FDA 5072 필수 항목
| 항목 | 내용 |
|---|---|
| Registration number | FDA Establishment Registration 번호 |
| Product identification | Brand명, Model, Lot, 510(k)/PMA 번호 |
| Event description | 결함 발생 경위 및 발견 날짜 |
| Risk evaluation | 부상·사망 발생 여부, MDR 번호 연계 |
| Distribution information | 직접 계약처(consignee) 수 및 국내외 배포 수량 |
| Proposed strategy | 회수 전략 (깊이, 효과성 점검 수준) |

### 6.3 MFDS — 회수계획서 필수 항목
| 항목 | 내용 |
|---|---|
| 회수 사유 | 결함 내용 및 근거 |
| 대상 제품 | 품목명, 허가번호, LOT 번호 |
| 회수 범위 | 제조·수입·유통 단계별 |
| 회수 방법 | 우편·이메일·현장 방문 등 |
| 회수 기한 | 완료 목표일 |
| 회수 후 처리 | 반환·폐기·수리 방법 |

## 8. 작성 체크리스트

### FSCA 착수 시
- [ ] 영향 제품·LOT 특정 완료
- [ ] 위험 평가 문서 작성 (Risk Assessment)
- [ ] 경영진 FSCA 착수 승인
- [ ] 3개 지역 규제 기관 보고 기한 확인
- [ ] FSCA 추적 대장 개설

### FSN 발행 시
- [ ] FSN 초안 작성
- [ ] EU: 관할 CA 사전 검토 요청
- [ ] FDA: DRC 사전 협의
- [ ] MFDS: 회수계획서 제출
- [ ] 최종 FSN 승인 (RA + 경영진)
- [ ] 모든 고객·유통망에 FSN 발송
- [ ] 발송 확인 기록 보관

### 진행 중
- [ ] 회수 진행 현황 주기적 추적 (2~4주 간격)
- [ ] 효과성 점검 결과 기록
- [ ] 규제 기관 진행 보고 (요청 시)

### FSCA 종료 시
- [ ] 최종 회수율 집계
- [ ] 종료 보고서 제출 (EU/FDA/MFDS)
- [ ] Root Cause Analysis 완료
- [ ] CAPA 개시 여부 결정
- [ ] PSUR/PMS 파일 반영
- [ ] FSCA 추적 대장 종료 기록
