# 기술문서 템플릿

## 수록 예정 템플릿
- **STED 기술문서**: ASEAN·아시아·국내 공통 요약형
- **Risk Management File (ISO 14971)**: Plan, Analysis, Evaluation, Control, Report
- **Clinical Evaluation Report (EU MDR)**: MDCG 2020-5 기반
- **Declaration of Conformity**: MDR Annex IV, MFDS, FDA 별
- **Labeling / IFU**: ISO 15223-1 기호, 다국어 IFU 양식

## 사용 원칙
- 템플릿은 제품 독립적인 Skeleton만 유지.
- 실제 案件 수행 시 `02_제품별_기술파일/` 또는 `03_진행현안/` 내 案件 폴더로 복사하여 사용.
- 템플릿 업데이트(예: 규제 개정 반영) 시 버전 관리 (파일명 말미 `_vX.X`).
