# SOP – Supplier Control (공급업체 관리) 절차서 템플릿

**문서 번호**: SOP-QMS-011  
**버전**: 1.0  
**기준 조항**: ISO 13485:2016 §7.4 (구매 프로세스)  
**작성일**: YYYY-MM-DD  
**검토자**: RA Manager  
**승인자**: QMS 대표자  

---

## 1. 목적 및 범위

### 1.1 목적
구매 제품·서비스가 규정된 품질 요구사항을 충족하도록 공급업체를 선정·평가·재평가·관리하고 승인 공급업체 목록(ASL)을 유지한다.

### 1.2 적용 범위
- X-ray Detector / Handheld X-ray Source / 촬영실 GUI SW에 사용되는 모든 외부 공급 부품, 원자재, 반제품, 서비스
- 위탁 제조업체 (Contract Manufacturer), 시험기관, SW 외주 개발사 포함

---

## 2. 용어 정의

| 용어 | 정의 |
|---|---|
| Approved Supplier List (ASL) | 품질 요건을 충족하여 구매가 허용된 공급업체 목록 |
| Critical Component | 제품 안전성·성능에 직접 영향을 미치는 부품 또는 원자재 |
| Supplier Qualification | 공급업체가 요구사항을 충족할 능력을 평가하는 프로세스 |
| Supplier Audit | 공급업체 현장 방문 또는 서류 검토를 통한 적합성 확인 |
| Incoming Inspection | 입고 제품의 요구사항 충족 여부 검사 |

---

## 3. 공급업체 분류 (위험 기반)

| 등급 | 기준 | 예시 | 관리 수준 |
|---|---|---|---|
| Class A — Critical | 안전성·성능에 직접 영향; 대체 어려움 | FPD 센서, X-ray 관구, 고압 발생기 | 연 1회 이상 현장감사; 적격성 시험 |
| Class B — Major | 제품 품질에 영향; 대체 가능 | PCB, 전원부, 케이스 | 문서 심사 + 연 1회 성과 평가 |
| Class C — Standard | 범용 자재; 쉽게 대체 가능 | 나사, 포장재, 소모품 | 입고 검사만 |
| 서비스 공급업체 | 시험기관, 멸균, SW 외주, 임상 CRO | 인증시험기관, IEC 62304 외주 개발사 | 별도 서비스 공급업체 평가서 |

## 4. 절차

### 4.1 신규 공급업체 선정 및 적격성 평가 (§7.4.1)

**평가 항목:**

| 평가 영역 | Class A | Class B | Class C |
|---|---|---|---|
| 재무 안정성 | 필수 | 필수 | 권고 |
| QMS 인증 (ISO 9001/13485) | 필수 | 권고 | — |
| 품질 이력 | 필수 | 필수 | 권고 |
| 현장 감사 | 필수 | 상황에 따라 | — |
| 샘플 승인 (FAI) | 필수 | 필수 | 입고 검사로 대체 |
| 기술 능력 평가 | 필수 | 필수 | — |

**평가 절차:**

```
[구매 요청] → [공급업체 후보 식별]
    → [자격 조사 (Desk Review)] → [현장 감사 (Class A)]
    → [FAI / 샘플 승인] → [평가위원회 승인]
    → [ASL 등재] → [구매 개시]
```

### 4.2 구매 정보 명시 (§7.4.2)

모든 발주서(PO) 또는 구매 사양서에 다음을 명시한다:

- 제품·서비스 정확한 사양 (도면, 규격서, 표준 번호)
- 승인 요건 (품질 검사 방법·기준)
- QMS 요건 명기 (해당 시 ISO 13485·KGMP 준수 요구)
- 규제 요건 (해당 시 FDA, MDR, MFDS 인허가 부품 요건)
- 기록 유지 요건

### 4.3 입고 검사 (§7.4.3)

| 등급 | 검사 수준 |
|---|---|
| Class A | 전수 검사 또는 고수준 샘플링 (AQL 1.0 이하) |
| Class B | 표준 샘플링 (AQL 2.5) |
| Class C | 육안 확인 또는 면제 (이력 기반) |

> 공급업체 성과가 우수한 경우 (3년 이상 NC 0건) 검사 수준 완화 가능 — 기록 근거 필요.

### 4.4 공급업체 재평가 (§7.4.1)

| 등급 | 재평가 주기 | 트리거 재평가 |
|---|---|---|
| Class A | 연 1회 | NC 발생, Recall, 주요 변경, 공급 차질 |
| Class B | 2년 1회 | Major NC, 반복 납기 지연 |
| Class C | 3년 1회 | 심각한 품질 문제 |

**성과 평가 지표 (Supplier Scorecard):**

| KPI | 가중치 | 측정 방법 |
|---|---|---|
| 품질 (NC율) | 40% | 입고 NC 건수 / 총 입고 수 |
| 납기 준수율 | 30% | 기한 내 납품 건수 / 총 발주 수 |
| 대응 속도 (불만 처리) | 20% | 평균 응대 시간 (영업일) |
| 문서 완결성 | 10% | 요청 문서 적시 제출율 |

**재평가 결과 조치:**

| 점수 | 등급 | 조치 |
|---|---|---|
| 85점 이상 | Preferred | ASL 유지, 우선 발주 |
| 70~84점 | Approved | ASL 유지, 모니터링 강화 |
| 50~69점 | Conditional | 개선 계획 요청, 6개월 재평가 |
| 50점 미만 | Disqualified | ASL 삭제, 대체 공급업체 확보 |

## 5. 기록 요건

| 기록 | 보존 기간 |
|---|---|
| ASL (Approved Supplier List) | 현행 유지 + 변경 이력 15년 |
| 공급업체 적격성 평가서 | 거래 종료 + 15년 |
| 현장 감사 보고서 | 동일 |
| 입고 검사 기록 | 제품 수명 + 15년 |
| Supplier Scorecard | 5년 |
| 공급업체 CAPA 요청 및 응답 | 제품 수명 + 15년 |

## 7. 자사 3개 제품 영향 맵핑

| 제품 | 핵심 공급업체 예시 | 특이 고려사항 |
|---|---|---|
| X-ray Detector | FPD 센서 제조사 (Varex, Teledyne 등), ASIC 공급사, 신틸레이터 업체 | Class A 지정; 센서 설계 변경 시 재적격성 평가 필수; 대체 공급업체 미확보 시 단일 소스 리스크 관리 (ISO 14971) |
| Handheld X-ray Source | X-ray 관구 (Varex, IAE 등), 고압 발생기, 배터리 셀 (IEC 62133-2) | 배터리 공급업체: UN 38.3·IEC 62133-2 인증 보유 확인 필수; 관구 교체 시 FDA PMA/510(k) 변경 검토 필요 |
| 촬영실 GUI SW | SW 외주 개발사, DICOM 라이브러리 공급사, OS 벤더 | IEC 62304 §5.3 SW 개발 계획에 외주 관리 포함; SOUP(기성 SW) 목록 관리; 사이버보안 취약성 모니터링 의무 |
